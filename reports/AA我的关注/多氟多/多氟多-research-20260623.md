## 第二步：生意本质分析 — 段永平"对的生意"

### 一句话定义

多氟多的核心生意是：**把萤石（CaF2）变成氟化工产品，其中最赚钱的是锂电池电解液核心材料六氟磷酸锂（LiPF6）**。本质上是一家资源加工型化工企业，产品价格由供需决定，企业几乎没有定价权。

### 商业模式画布

```
上游原料          多氟多加工            下游客户
━━━━━━━━━━      ━━━━━━━━━━━━━      ━━━━━━━━━━━━
萤石(CaF2)  →   氟化氢(HF)      →   氟化铝 → 铝企
氢氟酸      →   LiPF6合成       →   电解液厂(天赐/新宙邦)
碳酸锂      →   NaPF6合成       →   钠电池厂
              ↓
           锂电池制造          →    新能源车/储能
           电子级化学品        →    半导体企业
```

**关键观察**：多氟多处于产业链中游，向上受制于萤石和碳酸锂价格，向下受制于电解液厂商的采购议价。LiPF6是标准化工品，不存在品牌溢价。

### 收入漏斗分析

**LiPF6业务**（占收入33%，但可能贡献60%+的利润波动）：

- 产能：6.5万吨/年（全球第二，份额20-21%）
- 2022年峰值价格：~60万元/吨 → 单吨毛利可能超过40万
- 2024年谷底价格：~5.4万元/吨 → 单吨亏损
- 2026年当前价格：~10-12万元/吨 → 单吨毛利估计3-5万
- 价格弹性极大：价格每变动1万元/吨 → 利润变动约6.5亿（6.5万吨×1万）

**氟化铝/冰晶石业务**（占收入30%）：

- 全球第一，市场份额领先
- 需求与铝冶炼挂钩，相对稳定
- 毛利率稳定但不高，主要提供现金流底座

### 5年盈利能力趋势图

```
净利润(亿):    15.99 → 31.92 → 5.62 → -3.08 → 2.13  [2026Q1年化约15亿]
                ↑         ↑        ↓       ↓↓       ↑
              景气上行   周期顶    价格暴跌  谷底    复苏中

毛利率(%):    28.5 → 35.2 → 16.8 → 10.2 → 14.0  [2026Q1: 27.6%]
```

### 段永平式追问

> 段永平常问："这个生意，如果你把它关掉十年再重开，还能赚钱吗？"

**回答**：这个问题对多氟多特别残酷。LiPF6生产需要持续运营的化工装置、合格的技术人员和稳定的供应链关系。关掉十年，装置腐蚀报废、团队流散、客户转移——重开基本等于新建。而且十年后固态电池可能已经普及，LiPF6需求可能大幅萎缩。

**但另一方面**：氟化铝/冰晶石业务绑定铝冶炼行业，只要还有铝就需要这些产品。这块基本盘确实能在十年后重开。

**生意质量评分：★★★**

理由：资源加工型化工企业，有技术壁垒（LiPF6纯度要求高），有规模优势（全球前二），但本质上是价格接受者，周期波动极大，不是段永平所说的"好生意"（高毛利、强品牌、消费者心智占领型）。

## 第三步：护城河评估 — 巴菲特"经济护城河"

> 巴菲特："在商业世界中，我寻找的是有着坚不可摧护城河的经济城堡。"

### 五类护城河逐一评估

#### 1. 品牌护城河 — ★☆☆☆☆ (几乎不存在)

LiPF6是化工原料，客户（电解液厂）只关心纯度、价格和交期，不存在品牌溢价。多氟多在铝用氟化物领域有行业知名度，但这不构成定价权。

#### 2. 网络效应 — ★☆☆☆☆ (不适用)

化工品生意没有网络效应。客户越多不会让产品变得更好或更便宜。

#### 3. 转换成本 — ★★☆☆☆ (较低)

LiPF6产品存在认证周期（3-6个月），客户更换供应商需要重新验证。但由于产品标准化程度高，且下游电解液厂通常有2-3家供应商，转换成本有限。

**反面**：在LiPF6紧缺时期，客户愿意支付溢价锁定供应，但这只是供给紧张时的暂时现象，不是结构性转换成本。

#### 4. 成本优势 — ★★★★☆ (核心护城河)

这是多氟多最实质的竞争优势：

- **一体化产业链**：从萤石→氟化氢→LiPF6，自产中间体降低成本
- **规模效应**：6.5万吨/年产能全球第二，摊薄固定成本
- **副产品协同**：生产LiPF6的副产品可用于氟化铝/冰晶石，减少废物处理成本
- **地理优势**：河南焦作靠近萤石产区，运输成本低

**但另一方面**：天赐材料产能更大（约9万吨），成本优势可能更强。且在行业低谷时，成本优势不能阻止亏损——2024年即使有成本优势仍亏3亿。

#### 5. 规模/许可壁垒 — ★★★☆☆ (中等)

- 化工行业有环保审批门槛，新建产能需要1-2年
- LiPF6生产技术要求较高（无水环境、高纯度），新进入者需要技术积累
- 但2021-2022年高价吸引大量新产能，证明壁垒并非不可逾越

### 护城河趋势：收窄中

- LiPF6行业2021-2022年高利润吸引大量新产能建设
- 2024年全行业供过于求，价格崩至成本线以下
- 虽然2025-2026年部分高成本产能退出，但总产能仍高于需求
- 长期看，固态电池如果商业化，将从根本上削弱LiPF6护城河

### 巴菲特式追问

> 巴菲特："我宁愿以合理的价格买一家好公司，也不愿以便宜的价格买一家平庸的公司。"

**关键问题**：多氟多的成本优势护城河在行业上行期带来超额利润，在下行期只能减少亏损。这种"只在好时候有用"的护城河，是否值得巴菲特所说的"好公司"之名？

**数据说话**：过去5年ROE波动从36.4%到-3.4%，平均约13.4%。如果按周期均值看，ROE尚可，但波动本身就是风险。

### 空方论点详析

**1. 估值陷阱风险（最核心的空方论点）**

当前PE(TTM) 95倍看似很高，但空方真正担心的是：市场已经price in了2026年21亿净利润的预期（Forward PE 23倍）。如果LiPF6价格在2026年下半年走弱：
- 实际净利润可能只有12-15亿 → PE 33-41倍
- 股价可能需要下调30-40%来反映预期落空

**2. 周期股的"估值诅咒"**

> 彼得·林奇的经典警告："周期股PE最低的时候恰恰是最贵的时候，PE最高的时候反而是最便宜的时候。"

2022年多氟多净利32亿，PE约6-8倍，看似便宜——但那是周期顶部。2024年亏损，PE为负——但那可能是最佳买点。当前Forward PE 23倍到底是合理还是昂贵？取决于你认为周期处于什么位置。

**3. 现金流持续为负的警示**

2026Q1经营现金流-6.88亿。一家号称高景气度的公司，为何赚钱却不产生现金？可能原因：
- 应收账款增加（下游赊购）
- 存货积压（囤货赌涨价）
- 产能扩张的资本开支

无论哪个原因，都说明利润质量存疑。

### 周期股估值的特殊方法论

**核心原则：周期股不能简单用PE估值**

> 彼得·林奇："周期股在PE最低时最危险，PE最高时反而最安全。"

对多氟多，需要采用以下替代估值方法：

#### 方法1：周期均值利润法

取过去一个完整周期（2021-2025年）的平均净利润：
- 平均净利润 = (15.99 + 31.92 + 5.62 + (-3.08) + 2.13) / 5 = **10.52亿**
- 按周期均值PE 15-20倍：合理市值 = 158-210亿 → 合理股价 = 13.3-17.6元
- **当前497亿市值相对周期均值显著偏高**

#### 方法2：PB估值法（适用于重资产周期股）

- 当前BVPS 9.3元，PB 4.49倍
- 化工行业合理PB区间：1.5-3.0倍
- PB 1.5倍对应股价：13.95元
- PB 3.0倍对应股价：27.90元
- **当前股价41.80元远高于PB合理区间上限**

#### 方法3：峰值/谷值利润法

- 峰值净利润（2022年）：31.92亿，峰值PE应给8-12倍 → 市值255-383亿 → 股价21.4-32.2元
