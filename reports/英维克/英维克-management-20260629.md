## 一、创始人/CEO背景

### 1.1 董事长兼总经理：齐勇

| 项目 | 详情 |
|------|------|
| 出生年份 | 1968年（现年58岁） |
| 籍贯 | 内蒙古包头市昆都仑区 |
| 学历 | 西安交通大学工业自动化专业，硕士研究生（1993届） |
| 财富排名 | 2025年胡润百富榜第370位，身家175亿元 |
| 任职时间 | 2013年7月29日至今（创始人，2005年创立公司） |
| 2025年薪酬 | 148.77万元（2024年为113.8万元，同比+30.7%） |

**职业履历**：

1. **包头钢铁公司**（1993-约1996年）：西安交通大学毕业后分配回老家包钢工作，获得"铁饭碗"。在包钢期间积累了对热量产生与温度控制的基础理解。

2. **华为电气**（约1996-2001年）：90年代中期"下海潮"中辞去包钢工作南下深圳，通过社会招聘进入华为电气担任工程师，后升任部门负责人。在华为电气深耕精密温控与通信基站散热技术。

3. **艾默生**（2001-2005年）：2001年华为以7.5亿美元将华为电气（已更名安圣电气）出售给美国艾默生公司，齐勇随之进入艾默生工作约4年。在艾默生期间接触到国际化管理体系和更系统的技术视野，但因文化差异和管理理念分歧决定离开。

4. **创立英维克**（2005年至今）：2005年带领7名前艾默生同事辞职创业，成立深圳市英维克科技有限公司。创业初期条件简陋，租用300平方米厂房兼做办公室和实验室，使用二手实验设备。

**创业关键节点**：
- 2005年：创立英维克，瞄准通信基站散热痛点
- 2006年：累计完成300余次实验，推出中国首台自主研发的通信户外柜空调
- 2013年：成为中国联通集团一类供应商
- 2016年12月29日：英维克在深交所上市（IPO发行价18元，发行2000万股）
- 2021年：率先提出Coolinside全链条液冷解决方案，实现单机柜200kW批量交付
- 2022年：发布Coolinside液冷解决方案
- 2024年："厂到场"全场景方案
- 2025年："生到升"全周期解决方案，形成"三全"液冷方案

**评价**：齐勇属于典型的"华为电气-艾默生系"创业者（与汇川技术等同源）。个人风格极其低调，公开报道极少。从包钢到华为电气再到艾默生的经历，使其兼具国企的扎实功底、华为的技术嗅觉和外企的国际视野。连续20年深耕温控一个赛道，2011-2024年连续14年实现收入与利润双增长，执行力突出。

**结论：非常稳定** ★★★★★

- 齐勇、欧贤华、陈川自2013年公司成立以来一直任职至今（超过13年）

- 2025年换届为第五届董事会，仅更换独立董事，核心经营团队完全保留

**研发投入**：

### 4.4 监管处罚记录

**2024年11月：深圳证监局出具警示函**

事由：
- 2021年商誉减值测试时，有关业务收入及毛利率预测依据不充分
- 2022年商誉减值测试时，有关业务纳入资产组依据不充分
- 不符合《企业会计准则第8号——资产减值》第十一条、第十八条规定

处罚：
- 公司被采取**责令改正**的行政监管措施，要求30日内提交整改报告
- 董事长齐勇、财务总监叶桂梁被出具**警示函**

**评价**：这一事件暴露了公司在财务合规方面的不足。商誉减值测试不规范是A股常见问题，但对于一家志在千亿市值的公司而言，这种"粗糙"值得关注。好在问题已于2023年年报中完成整改，不影响正常经营。但它提示投资者：**管理层在财务审慎性方面存在短板**。

## 六、综合评估

### 管理层能力矩阵

| 维度 | 评分 | 说明 |
|------|------|------|
| 战略眼光 | ★★★★★ | 每一步都踩中行业拐点：通信→数据中心→液冷→储能→海外 |
| 执行能力 | ★★★★ | 14年连续双增长；但2026Q1利润暴跌暴露扩张过程中的管理短板 |
| 技术积累 | ★★★★★ | 182项发明专利，研发人员占比34%，全链条液冷唯一供应商 |
| 资本配置 | ★★★ | 研发投入大方但现金流管理粗糙，激进扩张导致财务结构脆弱化 |
| 财务纪律 | ★★★ | 商誉减值测试被监管处罚；应收账款和存货管理偏粗放 |
| 股东回报 | ★★★★ | 连续分红，管理层薪酬克制，利益与股东深度绑定 |
| 治理透明度 | ★★★★ | 定期披露，接受机构调研，但创始人极为低调 |
| 管理层稳定性 | ★★★★★ | 核心团队超过10年未变，创业元老团队凝聚力极强 |

### 主要风险提示

1. **"一人公司"风险**：齐勇身兼董事长+总经理+法人代表+实际控制人，权力过度集中。齐勇已58岁，公司尚无公开的接班人计划。

2. **财务管理能力跟不上业务增速**：2026Q1净利润暴跌82%，经营现金流-3.86亿元，短期借款激增136%。公司正从"内生增长"转向"融资输血"模式，如果液冷订单交付不及预期，流动性风险可能暴露。

3. **监管合规短板**：商誉减值测试违规被警示函，说明公司在财务合规方面存在不足。

4. **应收账款与存货持续高企**：2025年末应收账款约30亿元（占营收比重约50%），存货12亿元以上。回款效率是检验经营质量的核心观测点。

5. **机构大规模减持**：2026Q1基金持仓减半以上，散户大量涌入，筹码结构恶化。

### 正面亮点

1. 管理层持股约47%（广义口径），利益与股东深度一致
2. 核心团队稳定性极强，"华为电气-艾默生系"创业基因优秀
3. 战略前瞻性行业领先，从通信到液冷的每次转型均成功
4. 中国大陆唯一获得英伟达NPN Tier 1资质的液冷服务商
5. 薪酬水平克制，无过度关联交易
6. 三期股权激励覆盖305人，有效绑定核心人才
