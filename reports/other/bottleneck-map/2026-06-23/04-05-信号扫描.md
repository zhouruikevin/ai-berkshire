## 硝化棉/棉短绒瓶颈详解

### 供应链拆解

```
Layer 0：155mm炮弹 / 火箭弹药
Layer 1：无烟火药（Smokeless Powder）/ 推进剂
Layer 2（重点）：硝化棉（Nitrocellulose）— 本次新发现的瓶颈节点
Layer 3：棉短绒（Cotton Linters）— 原料；中国控制欧洲供给>70%
Layer 4：棉花加工副产品供应
```

### 瓶颈判定6条标准

| 标准 | 评估 | 评级 |
|------|------|------|
| 供给集中度 | 全球棉短绒高纯来源：中国>70%（欧洲视角） | 🔴 |
| 扩产周期 | 新建硝化棉产能需3-5年（化工许可+建设） | 🔴 |
| 替代难度 | BAE开发无需硝化棉的推进剂，但技术成熟期2026年底，量产待定 | 🟡 |
| 产能利用率 | 欧洲存量产能已满负荷运行，扩产项目大量开工 | 🔴 |
| 需求增速 | 俄乌战争+欧洲国防预算膨胀，弹药需求持续加速 | 🔴 |
| 客户验证周期 | 军用推进剂替换须经国防部长期验证（1-3年） | 🔴 |
