## 信号 A：中国放宽InP出口管制（2026-06-15，今日新信号）

**来源**：Digitimes（China eases InP substrate exports lifting compound semiconductor supply 2026-06-15）、Futunn（DIGITIMES China approved batch InP substrate exports 2026-06-15）、Digitimes（China squeezes InP supplies jolting US-Taiwan optics chain 2026-06-11）、BigGo Finance（Indium Phosphide prices surge 250% China export controls 2026）

### 事件梗概

| 维度 | 内容 |
|------|------|
| **核心事件** | 中国于2026-06-15批准新批次磷化铟（InP）衬底出口许可；5月底首批2026年度货物已完成发货 |
| **背景** | 2025年2月起中国对InP实施出口管制；6英寸InP晶圆价格从$1,400飙升至~$5,000（+250%） |
| **促成原因** | 美中5月贸易谈判期间，Anderson访华时将InP出口许可证延迟列为重点议题，贸易谈判后中国做出部分让步 |
| **规模** | 目前仅为"批次"而非"全面解除"——许可制度仍在，但管控开始松动 |
| **比较基准** | 6/11，Digitimes仍报道"中国收紧InP供应，冲击美台光学链"；6/15转向"中国放宽" |

**与现有瓶颈逻辑的交叉验证**：

- 本系统对InP/EML S级瓶颈的核心假设之一是"中国InP出口管制 → 西方供给无法快速填补"
- 今日新信号显示：**中国出口管制已开始软化**，这是InP/EML S级瓶颈的重要"潜在降级信号"
- 但：（1）许可证制度仍在，非永久性解除；（2）美中贸易关系仍有不确定性；（3）Ras Laffan损毁导致的制造业材料整体供需偏紧仍在

### 对各标的的影响

**AXTI（AXT Inc，纳斯达克 ★★）**：
- 逻辑变化：AXTI子公司Tongmei是最大中国InP生产商，此前积压$100M+因出口管制无法转化为收入
- 今日信号意味着Q2/Q3 2026收入可能出现明显提速（积压开始可出货）
- 估值挑战不变：市值~$5.1B，PS ~40x，红灯仍在；但**基本面触底反转信号增强**
- 结论：★★维持，但关注Q2财报（7月下旬）——若收入大幅超预期+PS降至<30x → 升级候选

**IQE plc（LON: IQE ★★★★候选）**：
- IQE生产外延晶圆（需要InP衬底投入）；中国InP衬底放宽 → 材料成本可能改善
- 但：中国InP衬底同时可供中国内部EML激光器制造商使用 → 可能扩大中国厂商竞争
- 净效果不明确；IQE ★★★★候选评级不变，等8月中报确认>20%增速+EBITDA转正
- 注意：FY2025 EBITDA仅£3.2M（同比-60%），FY2026需大幅改善才能晋升

**VECO（Veeco，纳斯达克 ★★★）**：
- 中国InP放宽 → 短期内西方EML厂商扩产压力略减 → 对VECO设备需求节奏有轻微不确定性
- 但：Nvidia $4B锁定COHR/LITE产能至2027+，这些公司仍需要MOCVD设备扩产以完成承诺
- 净效果：VECO需求不受实质影响，★★★维持；等股价回调至$60-62区间

**InP/EML S级瓶颈整体评估**：

| 因素 | 变化方向 |
|------|---------|
| 中国InP出口许可（信号A） | ⬆️ 供给松动（但许可制仍在，非永久解除） |
| Coherent 6英寸晶圆+产能翻倍（信号C） | ⬆️ 西方供给中期扩张 |
| LightCounting：mid-2026缓解预期 | ⬆️ 市场共识看法 |
| Nvidia $4B锁定COHR/LITE至2027+ | ⬇️ 供给端仍被大客户垄断 |
| 800G光模块产能缺口40-60%（至2027） | ⬇️ 需求侧瓶颈仍在 |

**阶段性判断**：InP/EML S级瓶颈仍维持，但供给端多重改善信号已出现。如果中国InP出口持续放宽+Coherent 6寸晶圆规模化，2026年底前瓶颈可能从S级降至A级。这是一个**下行风险增加的S级瓶颈**，而非坚不可摧的单点故障。
