# 三生国健（688336）深度投资研究

> 研究日期：2026年5月16日
> 初筛评级：4星（科创板召回池）
> 定位：自免疫领域平台型生物药企业，辉瑞60亿美元授权合作创国产创新药首付款纪录

---

## 一、商业模式与竞争壁垒

### 1.1 主营业务拆解

三生国健成立于2002年，2020年7月登陆科创板，是中国第一批专注于抗体药物的创新型生物医药企业。公司是三生制药（01530.HK）控股子公司，实控人为娄竞。

**业务板块梳理：**

| 板块 | 主要产品/项目 | 2025年情况 |
|------|-------------|-----------|
| 成熟产品 | 益赛普（TNF-α抑制剂） | 集采压力下价格下降，销量有所增长 |
| 成长产品 | 赛普汀（CD20单抗）、健尼哌（IL-2R单抗） | 赛普汀增速放缓，健尼哌基数小增速快 |
| CDMO | 合同研发生产 | 约0.7亿/半年（2024H1），增速约60% |
| 里程碑收入 | 辉瑞707项目授权 | 2025年确认约28.9亿元首付款收入 |

**2025年全年业绩：**
- 营业总收入42.0亿元，同比增长251.8%
- 归母净利润29.0亿元，同比增长311.5%
- 扣非净利润27.7亿元，同比增长1025.0%

**核心问题：扣除辉瑞授权收入后，常规业务体量如何？**

- 辉瑞707项目首付款确认收入约28.9亿元
- 扣除后，常规业务营收约13.1亿元
- 对比2024年营收约11.9亿元，常规业务增长约10%
- 以2024H1数据推算（益赛普3.3亿+赛普汀1.6亿+健尼哌0.3亿+CDMO 0.7亿），常规产品年化约12-13亿

**结论：剥离辉瑞一次性收入后，常规业务体量约13亿元，增速约10%，体量偏小。**

### 1.2 辉瑞合作深度拆解

这是理解三生国健估值的关键。

**合作标的：707项目（SSGJ-707）**
- 靶点：PD-1/VEGF双特异性抗体
- 技术平台：基于公司自研CLF2平台开发
- 临床进展：2025年4月获CDE突破性治疗认定，加速一线NSCLC适应症开发
- II期数据：在NSCLC患者中ORR和DCR数据优异，安全性良好

**财务条款：**

| 项目 | 金额 |
|------|------|
| 首付款（不可退还） | 12.5亿美元（约90亿人民币） |
| 开发/监管/销售里程碑 | 最高48亿美元 |
| 销售分成 | 双位数百分比梯度分成 |
| 合计最高交易金额 | 60.5亿美元 |

**中国大陆选择权：**
2025年7月，辉瑞获得中国大陆独家开发和商业化707的选择权，期权金+行权金合计不超过1.5亿美元。

**权益分配：**
707项目在三生国健体内的权益比例约30%（其余70%归三生制药集团层面）。

**关键问题：后续里程碑能否兑现？**

- 48亿美元里程碑需要707在全球多个适应症、多个监管区域获批并达到特定销售额才能全部触发
- 首款适应症（一线NSCLC）面临激烈竞争：阿斯利康Imfinzi+化疗、默沙东Keytruda+化疗等已是标准治疗方案
- PD-1/VEGF双抗赛道竞争者众多（阿斯利康的AZD2936、礼来/信达的IBI363等）
- 乐观情景下，后续里程碑可能分批在3-8年内逐步兑现
- 悲观情景下，如果III期临床数据不及预期，后续里程碑可能大幅缩水

### 1.3 自免管线评估

公司自2021年战略转型，聚焦自身免疫性疾病领域，目前在研22个自免项目。

**核心管线进展：**

| 项目代号 | 靶点 | 适应症 | 进展 |
|---------|------|--------|------|
| 608 | IL-17 | 银屑病 | 2024年11月申报NDA |
| 610 | IL-5 | 成人哮喘 | III期入组中 |
| 611 | IL-4Rα | 特应性皮炎 | III期主要终点已读出，预计2025H2申报NDA |
| 安沐奇塔单抗 | — | — | 2026年2月获批上市 |

**管线评价：**

- 正面：管线覆盖面广，22个自免项目形成梯队，608和611均接近商业化
- 但另一方面：靶点多为已验证的"me-too"或"me-better"靶点（IL-17、IL-4Rα、IL-5），面临度普利尤单抗（达必妥）、司库奇尤单抗等重磅药物的竞争
- 自免领域商业化需要庞大的学术推广团队和患者教育投入，三生国健的商业化能力尚待验证

### 1.4 公司治理关注点

三生国健与母公司三生制药（01530.HK）之间存在大量关联交易和资产注入历史。媒体曾报道"三次左手倒右手"的运作模式。707项目的权益仅30%归属三生国健，大头在母公司层面。需关注关联交易的公允性和少数股东权益保护。

## 二、财务质量

### 2.1 盈利能力分析（剥离辉瑞一次性收入）

| 指标 | 2023年 | 2024年 | 2025年（常规） | 2025年（含辉瑞） |
|------|--------|--------|---------------|----------------|
| 营业收入 | 约9.7亿 | 约11.9亿 | 约13.1亿（估计） | 42.0亿 |
| 归母净利润 | 约1.7亿 | 约7.0亿 | — | 29.0亿 |

- 常规业务收入约13亿元，增速约10%，对应400亿市值的PS约31倍
- 扣除辉瑞授权后的常规净利润难以准确计算（因为费用结构混合），但考虑到高研发投入（研发费用率约32%），常规业务的净利润体量可能仅2-4亿元
- 2026Q1归母净利润1.78亿元，同比增长73%，但这是在2025Q1低基数上的增长

### 2.2 研发投入

2025Q1研发费用同比增长45.83%，研发投入占营收比重32.20%。22个在研项目需要大量持续投入，这是把"双刃剑"——既是未来管线的基础，也意味着短期利润承压。

### 2.3 现金流与资产质量

辉瑞首付款12.5亿美元（约90亿人民币，其中约28.9亿归属三生国健确认为收入）带来了充裕的现金。但这是一次性收入，需关注：
- 后续研发投入节奏是否会显著加大
- 是否有分红或回购计划将现金回馈股东
- 母公司层面的资金调配是否影响上市公司利益

## 三、成长性与估值

### 3.1 增长驱动力

**短期（1-2年）：**
- 608（IL-17银屑病）和611（IL-4Rα特应性皮炎）获批上市后的商业化放量
- 安沐奇塔单抗（2026年2月已获批）的销售起步
- 707项目后续里程碑触发的可能性（取决于全球临床进展）

**中期（3-5年）：**
- 自免管线中多款产品陆续获批，从单品公司向平台型公司转型
- 707项目如果全球获批上市，销售分成将带来持续收入流

**远期：**
- 如果自免平台验证成功，更多创新靶点进入临床
- 707全球销售分成的长期贡献

### 3.2 估值判断

| 指标 | 数值 |
|------|------|
| 当前市值 | 约412亿元 |
| 2025年归母净利润（含辉瑞） | 29.0亿元 |
| 表观PE（含辉瑞） | 约14倍 |
| 常规业务估计净利润 | 2-4亿元（估计） |
| 常规业务PE | 100-200倍（估计） |
| 动态PE（2026年3月数据） | 约66倍 |

**估值的核心矛盾：**

- 表面看PE仅14倍，非常便宜。但辉瑞28.9亿首付款是一次性的，不可持续。
- 剥离辉瑞收入后，常规业务PE高达100-200倍，说明市场对管线给予了极高的期望值。
- 412亿市值对应的是"平台型自免公司"的预期估值，而非当前业务的估值。
- 关键假设检验：608、611等新品能否在3年内贡献10亿+收入？考虑到竞品众多（度普利尤单抗在中国销售已达数十亿级别），这个目标有挑战。

**结论：当前估值反映了辉瑞合作后市场对公司创新能力的重新定价，但常规业务体量偏小、管线多为跟随靶点、商业化能力未经验证，412亿市值包含了大量预期。适合长期跟踪管线进展，不宜追高。**

## 四、风险评估

### 4.1 核心风险

| 风险因素 | 严重程度 | 说明 |
|---------|---------|------|
| 辉瑞合作不可持续性 | ★★★★★ | 28.9亿首付款是一次性收入，后续里程碑触发高度不确定 |
| 管线临床失败风险 | ★★★★☆ | 22个在研项目，每个都有临床失败的可能 |
| 商业化执行风险 | ★★★★☆ | 自免领域需要强大的商业化团队，公司历史上商业化能力一般 |
| 关联交易风险 | ★★★☆☆ | 与母公司三生制药存在大量关联交易，707权益仅30% |
| 益赛普集采风险 | ★★★☆☆ | 基本盘产品面临集采降价，收入持续承压 |
| 竞品风险 | ★★★★☆ | 自免领域竞争激烈，度普利尤单抗等先发优势明显 |

### 4.2 正面因素

- 辉瑞合作验证了公司的创新能力和分子设计水平
- 22个自免管线构成了较好的产品梯队
- 707项目如果成功，销售分成可带来长期持续收入
- 2026年2月安沐奇塔单抗获批，管线兑现能力初步验证

## 五、综合结论

**投资吸引力评分：★★★☆☆（中性，需跟踪管线兑现）**

三生国健因辉瑞60亿美元授权合作而"一战成名"，市场对其创新能力给予了重新定价。但冷静分析后，几个核心问题值得深思：

1. **常规业务偏小**：剥离辉瑞收入后，年营收仅约13亿，对应400亿+市值明显偏贵。
2. **辉瑞合作的"含金量"分析**：12.5亿美元首付款中仅约28.9亿人民币归属三生国健（约30%权益），大头在母公司。
3. **管线以跟随靶点为主**：IL-17、IL-4Rα等均为已验证靶点的me-too/me-better，差异化竞争力有待临床数据验证。
4. **从"卖药"到"license-out"的估值逻辑切换**：市场正在用biotech平台估值法（管线价值叠加），而非传统PE估值，这增加了估值的主观性。

公司如果能在未来2-3年内实现608、611等产品的成功商业化，同时707海外临床进展顺利触发新的里程碑付款，估值有望获得基本面支撑。反之，则面临估值回调风险。

### 关键跟踪指标

1. **608（IL-17银屑病）NDA审评进度和获批时间**
2. **611（IL-4Rα特应性皮炎）NDA申报和审评**
3. **707项目全球III期临床数据和进展**
4. **季度常规产品收入增速**：益赛普、赛普汀等是否企稳
5. **辉瑞后续里程碑触发情况**
6. **关联交易公允性**：与三生制药的交易条款是否对上市公司有利
7. **研发费用率变化**：22个项目的投入节奏和资金使用效率
