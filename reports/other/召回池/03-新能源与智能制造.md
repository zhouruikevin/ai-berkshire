## 比亚迪股份（1211.HK）

### 核心业务

比亚迪是全球新能源汽车龙头，主营新能源汽车（含纯电动和插电混动）、动力电池、半导体及消费电子代工（通过比亚迪电子）。2025年汽车销量突破427万辆，其中新能源车占比超过97%，海外销量105万辆（同比增长150%）。收入构成上，汽车及相关产品贡献约75%营收，手机部件及组装业务贡献约20%。比亚迪是全球少数实现从电池原材料到整车制造垂直整合的车企，约75%核心零部件自产，这赋予其相较欧美竞争对手约25%的成本优势。公司在全球新能源车市场市占率持续领先，2025年全年销量全球第一。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约8,400亿港元（估计） |
| PE(TTM) | 约26倍（估计，基于港股股价98港元） |
| ROE | 约15%（估计） |
| 营收（2025） | 8,040亿元 |
| 营收增长率 | +3.5% |
| 净利润（2025） | 326亿元 |

| 自由现金流 | 约854亿元（机构预测） |

### 护城河分析

**正面：** 比亚迪的护城河核心在于垂直整合与规模效应的叠加。公司自研自产电池（刀片电池）、电驱系统、IGBT芯片、车载半导体等核心部件，上游到中游双重卡位，构建了极难复制的产业壁垒。研发投入常年超过净利润，2025年研发投入接近500亿元，超过三大传统车企研发投入之和。规模效应方面，427万辆年销量带来的采购成本摊薄、产能利用率提升、售后网络密度都在不断强化成本壁垒。海外扩张也在加速，105万辆海外销量已开始构建全球化品牌认知。

**反面：** 王传福自己也坦承"技术领先度不及以往"，竞争对手在智驾、座舱体验上追赶速度加快。2025年"增收不增利"（营收增3.5%但净利润降19%）暴露了价格战对盈利能力的侵蚀。4.2%的净利率在制造业中偏低，护城河的宽度取决于是否能从"成本领先"升级为"品牌溢价"。70.7%的资产负债率虽然大部分是经营性负债（对供应商的议价体现），但重资产模式在行业下行期风险不小。

### 管理层简评

王传福是比亚迪的灵魂人物，典型的技术型创始人，从电池到汽车再到半导体，战略眼光一流。巴菲特当年投资比亚迪的核心逻辑就是"投人"，王传福信守承诺的品格被视为投资基石。管理团队执行力极强，从DM-i到刀片电池到海豹等产品的快速迭代体现了组织效率。但公司高度依赖王传福个人决策，接班人梯队尚不清晰，这是长期持有者需要关注的风险。

### 估值判断

港股约98港元对应市值约8,400亿港元，按2025年净利润326亿元计算PE约26倍，对于一家净利润下滑19%的制造企业来说不算便宜。但如果2026年机构预测的447-470亿净利润能兑现，对应前瞻PE约18-19倍，考虑到全球新能源龙头地位，估值合理偏中性。关键变量是2026年价格战是否缓解、海外市场能否贡献利润增量。

### 初步评级

- 生意质量：★★★☆☆（汽车制造是苦生意，净利率仅4.2%）
- 护城河强度：★★★★☆（垂直整合+规模效应强大，但技术领先被追赶）
- 管理层质量：★★★★★（王传福顶级，但单人依赖风险）
- 估值吸引力：★★★☆☆（不贵不便宜，需要利润增长兑现）
- 综合评分：★★★★☆
- 是否值得深入研究：是
- 一句话总结：全球新能源车王者，护城河够宽但生意本质是制造业苦差事，净利率说明一切。

## 理想汽车（2015.HK）

### 核心业务

理想汽车专注家庭用户市场的智能电动车，主力产品为L系列增程式SUV（L6/L7/L8/L9）以及新推出的纯电车型。2025年全年交付40.63万辆（同比下滑约20%），营收1,123亿元。理想是新势力中首家营收破千亿的公司，也是连续三年保持盈利的新势力。商业模式以直营为主，产品定位精准瞄准"有娃家庭"需求，单车均价约27-28万元，在25-50万元SUV市场占据头部地位。AI相关研发投入占比50%，全栈自研智驾系统是公司重点投入方向。

### 关键财务指标

|------|------|
| 市值 | 约1,600亿港元（估计，基于港股77港元） |
| PE(TTM) | 约145倍（基于净利11亿估算） |
| ROE | 约1%（估计，利润极薄） |

### 护城河分析

**正面：** 理想的产品定义能力是其最核心的竞争力——精准锁定"奶爸车"细分市场，L系列产品功能配置与有娃家庭使用场景高度匹配，形成了极高的用户认知度。技术方面，理想实现芯片、编译器、操作系统、基座模型的全栈自研，VLA智驾大模型持续迭代。1,012亿现金储备提供了充足的战略缓冲。管理层重构了研发组织，采用扁平化架构推动AI成为核心生产力。

**反面：** 2025年销量和利润双降暴露了产品周期的脆弱性——当L系列进入产品末期，增长就会失速。增程式技术路线在纯电趋势面前面临被边缘化风险，纯电新品能否成功是巨大的不确定性。管理层自己都说"不迷信护城河"，这种坦诚反过来也说明理想的护城河确实不深——本质上是产品驱动型公司，缺乏结构性壁垒。高管出走问题也值得关注。

### 管理层简评

李想是少见的具有极强产品感知力的创始人，从理想ONE到L系列的产品定义堪称教科书。但2025年销量下滑暴露了组织管理的短板——公司一度出现战略摇摆（MEGA失利后的调整期）。管理层坦承不迷信护城河、注重用户价值，这种务实态度值得认可。2026年"3+2"战略（3款增程+2款纯电）是关键考验，能否从"爆款驱动"转向"体系化能力"将决定公司天花板。

### 估值判断

港股约77港元对应市值约1,600亿港元，按2025年净利润11亿元计算，PE高达145倍，显然当前利润基数太低不具参考性。市场定价的是2026年销量恢复至48万辆、利润回升的预期。如果2026年能回到50亿以上净利润，对应约32倍PE，对于成长型车企尚可接受。但不确定性极高，估值需要持续跟踪。

### 初步评级

- 生意质量：★★★☆☆（连续盈利但2025年利润近乎归零）
- 护城河强度：★★☆☆☆（产品驱动，缺乏结构性壁垒）
- 管理层质量：★★★★☆（产品力强，但组织管理有待验证）
- 估值吸引力：★★☆☆☆（高PE需要利润大幅反弹兑现）
- 综合评分：★★★☆☆
- 是否值得深入研究：是（关注纯电产品成败）
- 一句话总结：产品定义一流但护城河浅，2025年的滑坡说明单靠爆款不可持续，需要证明体系化能力。

## 小鹏汽车（9868.HK）

### 核心业务

小鹏汽车定位智能电动车，以智能驾驶为核心卖点。2025年全年交付42.9万辆（同比增长126%），营收767.2亿元（同比增长87.7%），Q4首次实现单季度盈利（净利3.8亿元）。主力产品为MONA M03（走量爆款）、P7+、G6等。商业模式除了自有品牌车型外，还向大众汽车输出智驾和电子电气架构技术（技术授权收入）。小鹏自研了图灵AI芯片（2250 TOPS算力）和鲲鹏超级电动体系，走"智驾+性价比"双路线。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约1,100亿港元（估计，基于港股约61港元） |
| PE(TTM) | 亏损状态（全年净亏11.4亿元，Q4单季盈利） |
| ROE | 负值（累计亏损） |
| 营收（2025） | 767.2亿元 |
| 营收增长率 | +87.7% |
| 净利润（2025） | -11.4亿元（亏损收窄80.3%） |
| 净利润增长率 | 亏损大幅收窄 |
| 毛利率 | 18.9%（Q4达21.3%创新高） |
| 净利率 | -1.5%（接近盈亏平衡） |
| 资产负债率 | 数据待确认 |
| 自由现金流 | 数据待确认 |

### 护城河分析

**正面：** 小鹏的智驾技术是其最明确的差异化标签。自研图灵AI芯片（对标特斯拉FSD芯片路线）、VLA端到端大模型、鲲鹏超级电动体系，构成了技术护城河。与大众汽车的深度合作（共同开发CEA架构）不仅带来技术授权收入，更是对小鹏技术实力的背书。MONA M03的成功证明小鹏开始具备"技术+爆款"的能力组合，Q4毛利率达21.3%说明规模效应开始显现。

**反面：** 全年仍然亏损11.4亿元，虽然亏损大幅收窄但盈利能力尚未稳固。小鹏过去几年经历了产品定位摇摆、组织架构大调整，执行力曾被严重质疑。智驾技术虽然领先，但在用户购买决策中权重有多大仍有争议——MONA M03的成功更多靠性价比而非智驾溢价。技术授权模式能否持续贡献收入也有不确定性。

### 管理层简评

何小鹏是技术型创始人，对AI和智驾的投入方向判断正确。但小鹏的管理团队在过去几年经历了大换血，顾宏地（总裁）的加入提升了商业化能力。2025年的全面变革（组织、产品、营销、渠道补短板）效果开始显现，但持续性有待验证。何小鹏将公司定位为"技术公司而非汽车公司"，这个野心需要用利润来证明。

### 估值判断

港股约61港元对应市值约1,100亿港元，当前仍处亏损状态无法用PE衡量。如果2026年能实现全年盈利（机构预测约20-30亿元），对应约37-55倍PE，以成长性看尚可接受但确实偏贵。小鹏的估值更多反映的是"AI+智驾"概念溢价和技术授权模式的想象空间。

### 初步评级

- 生意质量：★★☆☆☆（刚到盈亏平衡线，盈利模式未验证）
- 护城河强度：★★★☆☆（智驾技术领先，但变现路径不清晰）
- 管理层质量：★★★☆☆（技术眼光好，执行力曾有波动）
- 估值吸引力：★★☆☆☆（概念溢价高，需要利润兑现）
- 综合评分：★★★☆☆
- 是否值得深入研究：是（关注技术授权变现和2026年盈利）
- 一句话总结：智驾技术最突出的新势力，但技术能否转化为持续盈利仍是问号，投资逻辑更像科技股而非车企。

## 蔚来（9866.HK）

### 核心业务

蔚来定位高端智能电动车，旗下三品牌：蔚来（高端，40万以上）、乐道（中端，20-30万）、萤火虫（入门，约15万）。2025年全年交付32.6万辆（同比增长46.9%），营收847.9亿元（同比增长33.1%）。Q4首次实现单季度经营盈利（经营利润8.07亿元，净利润2.8亿元）。蔚来独创的换电网络是其核心差异化——已建成覆盖全国的换电站网络，并与多家车企合作开放换电。此外，蔚来还重投入自研芯片、电驱和智驾系统。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约1,081亿港元 |
| PE(TTM) | 亏损状态（全年净亏149.4亿元） |
| ROE | 负值（累计亏损超千亿） |
| 营收（2025） | 847.9亿元 |
| 营收增长率 | +33.1% |
| 净利润（2025） | -149.4亿元（亏损收窄33.3%） |
| 净利润增长率 | 亏损大幅收窄 |
| 毛利率 | Q4达17.5%（整车18.1%） |
| 净利率 | 约-17.6%（全年） |
| 资产负债率 | 约87%-93%（极高） |
| 自由现金流 | Q3/Q4连续正向，年末现金储备459亿元 |

### 护城河分析

**正面：** 蔚来的换电网络是独一无二的基础设施型护城河——建设门槛极高，竞争对手很难短期复制。用户社区（近50万用户的高粘性社群）和服务体验是另一道壁垒。三品牌战略覆盖15万-50万价格带，产品矩阵完整。李斌将研发投入累计近400亿元打造的技术体系（包括换电站、芯片、电驱）视为长期竞争力的根基。

**反面：** 累计亏损超千亿，资产负债率87%-93%，财务状况在所有新势力中最脆弱。换电网络到底是护城河还是包袱至今争论不休——前期投入巨大，回本周期极长，如果销量不能持续放量，换电网络的固定成本会成为巨大拖累。2025年全年仍亏损149亿元，虽然Q4盈利了但全年距离盈亏平衡还有很长的路。多品牌运营对组织能力的要求极高，萤火虫品牌的市场表现也有待观察。

### 管理层简评

李斌是中国造车新势力中最有韧性的创始人之一，蔚来多次在生死边缘挣扎又活了过来，李斌的融资能力和战略定力不容忽视。秦力洪（总裁）在用户运营方面表现出色。但管理层制定的目标过于激进（市值目标300亿-1200亿美元），与当前财务现实差距巨大。组织效率有提升空间——同样的销量规模，蔚来的亏损远超竞争对手。

### 估值判断

当前市值约1,081亿港元，仍处巨额亏损状态。管理层宣布2026年全年实现Non-GAAP盈利的目标，如果实现（假设20-30亿元调整后利润），对应约36-54倍PE。但考虑到累计亏损和资产负债率极高，估值风险不小。蔚来更适合作为"反转故事"来看，需要持续跟踪季度盈利改善趋势。

### 初步评级

- 生意质量：★★☆☆☆（重资产、亏损累累，盈利拐点刚出现）
- 护城河强度：★★★☆☆（换电网络独特但经济性存疑）
- 管理层质量：★★★☆☆（韧性强但执行效率偏低）
- 估值吸引力：★★☆☆☆（反转故事风险大）
- 综合评分：★★☆☆☆
- 是否值得深入研究：否（财务风险太高，生意模式未验证）
- 一句话总结：最有理想主义色彩的新势力，但千亿亏损和极高负债率让投资者很难安心入眠。

## 吉利汽车（0175.HK）

### 核心业务

吉利汽车是中国最大的民营汽车集团之一，2025年整合极氪后形成"大吉利"体系，旗下品牌包括吉利银河（新能源主力，年销124万辆）、领克（年销35万辆）、极氪（年销22.4万辆）以及传统燃油车。2025年全年销量302.46万辆（同比增长39%），营收3,452亿元（同比增长25%），市占率首次站上10%。商业模式涵盖乘用车、商用车、出行服务和飞行汽车等，是典型的多品牌、多技术路线布局。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约2,454亿港元 |
| PE(TTM) | 约5.9倍（基于核心净利144亿估算） |
| ROE | 约17.7% |
| 营收（2025） | 3,452亿元 |
| 营收增长率 | +25% |
| 净利润（2025） | 168.5亿元（归母）/ 144.1亿元（核心） |
| 净利润增长率 | 核心净利+36% |

港股PE约9.6倍，股息率高达6.36%，在港股家电板块中估值极具吸引力。对于一家ROE 17%、营收3,000亿元的全球化家电龙头，不到10倍PE明显偏低。港股相对A股存在显著折价，这为价值投资者提供了机会。增长虽慢但确定性高，加上持续提升的分红比例，适合追求稳健回报的投资者。

### 初步评级

- 生意质量：★★★★☆（全球化布局稳健，现金流好）
- 护城河强度：★★★★☆（全球化深度独一无二，品牌矩阵强）
- 管理层质量：★★★★☆（体系化管理，股东回报意识强）
- 估值吸引力：★★★★★（9.6倍PE+6.4%股息率，极具安全边际）
- 综合评分：★★★★☆
- 是否值得深入研究：是（港股低估的全球化家电龙头）
- 一句话总结：全球化最深的中国家电公司，港股不到10倍PE+6%股息率，是价值投资者的理想标的，代价是增长偏慢。

## 创科实业（0669.HK）

### 核心业务

创科实业（TTI）是全球充电式电动工具市场领导者，旗下核心品牌Milwaukee（美沃奇，贡献超60%收入，定位专业/工业级）、RYOBI（良明，定位DIY/消费级）、MILWAUKEE等13个子品牌覆盖消费者到工业全场景。2025年全年营收153亿美元（同比增长4.4%），归母净利润11.98亿美元（同比增长6.8%）。电动工具业务贡献144亿美元收入（占94%），地板护理及清洁业务贡献9亿美元。公司总部位于香港，生产基地遍布亚洲、欧洲、美洲，核心市场为北美（通过家得宝Home Depot渠道）。

### 护城河分析

**正面：** 创科实业的护城河是港股中最接近巴菲特标准的之一。第一层是品牌壁垒——Milwaukee在北美专业电动工具市场的搜索热度自2008年超过Dewalt后一路领先，品牌忠诚度极高。第二层是平台锁定效应——M12/M18/MX FUEL等电池平台兼容数百款工具，用户一旦进入平台就很难切换。第三层是渠道壁垒——与家得宝的独家合作关系延续20多年。第四层是创新飞轮——每年研发投入约7.6亿美元（占营收5%），持续推出新产品保持技术领先。41.2%的毛利率远超一般制造业，说明品牌溢价极强。

**反面：** 2025年营收仅增长4.4%，增速放缓。北美房地产市场降温可能影响工具需求。对家得宝渠道的依赖度过高，大客户集中度是风险点。地板护理及清洁业务持续拖累整体表现。关税和贸易摩擦可能影响亚洲生产基地到北美市场的成本结构。

### 管理层简评

创科实业的管理层团队极为稳定且具有战略远见。CEO Joseph Galli Jr拥有在Black & Decker超过19年的行业经验，对电动工具市场了如指掌。创始人Horst Pudwill家族保持战略层面的把控。从1999年上市到2007年间，管理层通过10次并购构建了如今的品牌帝国，其中2005年前瞻性布局锂电、收购Milwaukee的决策堪称神来之笔。公司自2007年起推出领导才能发展计划，3年从顶尖大学招聘超过4,000名学员，人才梯队建设扎实。

### 估值判断

市值约1,782亿港元，PE约23倍。2026年机构预测净利润16.3亿美元（同比增长21.3%），对应前瞻PE约19倍。对于一家ROE接近20%、毛利率41%、几乎零负债的全球龙头来说，23倍PE合理但谈不上便宜。创科实业长期是港股的"百倍股"传奇（上市以来股价涨超25倍），市场给予了品质溢价。等待回调至20倍PE以下可能提供更好的买入机会。

### 初步评级

- 生意质量：★★★★★（41%毛利率+20% ROE+零负债，顶级制造业）
- 护城河强度：★★★★★（品牌+平台+渠道+创新，四重壁垒）
- 管理层质量：★★★★★（战略远见+执行力+人才培养，一流）
- 估值吸引力：★★★☆☆（23倍PE合理但不便宜）
- 综合评分：★★★★★
- 是否值得深入研究：是（港股最优质的制造业资产之一）
- 一句话总结：港股版"丹纳赫"，全球电动工具绝对霸主，41%毛利率+零负债的生意质量在制造业中凤毛麟角，值得长期跟踪等待好价格。
