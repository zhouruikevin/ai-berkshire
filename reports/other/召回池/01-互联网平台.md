## 腾讯控股（0700.HK）

### 核心业务

腾讯是中国最大的互联网公司，业务覆盖社交通信、游戏、数字内容、金融科技、云计算和企业服务。核心产品微信/WeChat月活超14亿，是中国最大的超级应用，承载社交、支付、短视频（视频号）、小程序等全场景生态。收入构成方面，2025年全年营收7518亿元，其中增值服务（游戏+社交）占比约40%，金融科技及企业服务占比约31%，网络广告占比约29%。腾讯在游戏领域全球收入排名第一，拥有《王者荣耀》《和平精英》等长青产品，并通过投资Epic Games、Riot Games等构建全球游戏版图。广告业务受益于视频号商业化和AI精准投放持续高增。腾讯在中国互联网行业地位无可撼动，是唯一同时在社交、游戏、支付、云计算四大领域占据头部位置的公司。

### 护城河分析

腾讯的护城河是中国互联网行业中最宽最深的，核心来源于微信构建的社交网络效应。14亿月活用户及其社交关系链的迁移成本极高，这不是用钱能复制的壁垒。微信生态已从即时通讯扩展到支付、小程序、视频号、搜索，形成"用户-商家-开发者"的多边网络效应，越用越强。

游戏业务方面，腾讯同时拥有全球最大的游戏分发渠道和顶级自研能力，形成"渠道+内容"双重壁垒。金融科技依托微信支付的高频使用场景，理财、保险、信贷业务自然延伸。云业务虽不是最大，但依托腾讯生态的企业客户资源，增长稳健。

弱点在于：游戏行业受监管政策影响大，版号审批不确定性始终存在；视频号虽然增长快但面临抖音的强力竞争；国际化相对滞后，海外收入占比有限。此外，腾讯体量已极大，维持双位数增长的难度在加大。

### 管理层简评

马化腾是腾讯灵魂人物，技术出身，决策风格稳健务实。腾讯的管理层以职业经理人团队为主导，刘炽平、James Mitchell等高管在战略和投资领域表现优异。腾讯的管理文化强调"用户体验优先"和"开放连接"，在过去20多年中做出了多次正确的战略选择：从QQ到微信的自我革命、从封闭到开放生态的转型、视频号的及时布局。治理结构清晰，马化腾持股约8%但拥有较强影响力。腾讯在资本配置上也表现出色，大规模回购+分红+分拆减持投资组合，股东回报意识强。

### 估值判断

当前Non-IFRS PE约18倍，处于腾讯历史估值区间的中低位。考虑到14%的营收增速、24%的ROE、强劲的自由现金流和AI驱动的增长潜力，18倍PE对于这种质量的公司并不算贵。同行对比，苹果PE约30倍，Meta约25倍，腾讯相对便宜。但需考虑港股市场整体折价因素和地缘政治风险的持续影响。整体估值合理偏低。

### 初步评级

- 生意质量：★★★★★
- 护城河强度：★★★★★
- 管理层质量：★★★★★
- 估值吸引力：★★★★
- 综合评分：★★★★★
- 是否值得深入研究：是
- 一句话总结：中国互联网行业当之无愧的核心资产，生意质量一流，估值合理，长期持有首选。

## 阿里巴巴（9988.HK）

### 核心业务

阿里巴巴是中国最大的电商平台公司，同时是领先的云计算服务商。核心业务包括淘宝天猫（国内电商）、国际电商（速卖通、Lazada、Temu竞争市场）、阿里云（中国排名第一）、本地生活（饿了么、高德）、菜鸟物流等。2025财年（截至2025年3月）全年营收约9963亿元。收入结构中，淘天集团仍是最大收入来源，但增速放缓至低个位数；阿里云受益于AI需求增长加速，AI相关收入连续多个季度三位数增长；国际电商和本地生活处于投入期。阿里在中国电商市场的份额受到拼多多、抖音电商等的持续侵蚀，但仍是品牌电商的第一选择。

### 护城河分析

阿里的护城河正在经历深刻变化。传统优势在于：淘宝天猫作为最大的品牌电商平台拥有庞大的商家生态和消费者基础，菜鸟物流网络覆盖全球，支付宝/蚂蚁金服在金融科技领域有深厚积累。阿里云是中国最大的公有云平台，在AI时代凭借全栈自研能力（芯片、平台、大模型）构建了新的竞争壁垒，蔡崇信称其为"全球少数同时拥有强大AI和领先云计算的公司"。

但护城河也在被削弱：拼多多以低价策略抢走大量价格敏感型用户；抖音电商以内容驱动交易，改变了用户购物路径；淘宝闪购等即时零售布局面临美团、京东的激烈竞争。阿里核心电商业务仅个位数增长，反映出竞争压力之大。此外，云计算虽增长快但处于大规模投入期，自由现金流大幅下降，短期盈利能力承压。

### 管理层简评

2023年吴泳铭接任CEO后推动"用户为先、AI驱动"战略，蔡崇信担任董事会主席。吴泳铭是阿里"十八罗汉"之一，技术背景深厚，上任以来进行了大刀阔斧的改革：裁员6.6万人、聚焦核心业务、加大AI投入（三年超3800亿元）。管理层战略方向正确，但执行效果仍需时间验证。阿里历史上的治理问题值得关注：频繁更换高管、组织架构反复调整、蚂蚁金服上市受阻等事件影响了市场信心。马云退居幕后但仍有影响力，当前管理层格局尚处于磨合期。

### 估值判断

当前15倍PE（Non-GAAP），是中国大型互联网公司中估值最低的之一。若按分部估值法，阿里云、蚂蚁金服、菜鸟等单独拆出均有显著价值，整体可能存在"集团折价"。但低估值有其原因：核心电商增长乏力、竞争格局恶化、自由现金流下降、AI投入期盈利承压。与腾讯18倍、拼多多约22倍相比，阿里估值偏低反映了市场对其基本面的担忧。属于典型的"便宜但不确定是否值"的状态。

### 初步评级

- 生意质量：★★★★
- 护城河强度：★★★
- 管理层质量：★★★
- 估值吸引力：★★★★
- 综合评分：★★★★
- 是否值得深入研究：是
- 一句话总结：低估值和AI云的增长潜力有吸引力，但核心电商护城河被侵蚀是最大隐忧，需要持续跟踪吴泳铭的改革成效。

## 美团（3690.HK）

### 核心业务

美团是中国最大的本地生活服务平台，核心业务包括餐饮外卖、到店及酒旅（团购、点评）、即时零售（美团闪购/买菜）和新业务（社区电商、海外扩张Keeta等）。2025年全年营收3649亿元，核心本地商业收入2608亿元，新业务收入1040亿元。美团在外卖市场稳定保持60%以上的GTV份额，是绝对龙头。到店酒旅是现金牛业务，利润率高。美团的商业模式本质是通过即时配送网络连接消费者和商家，拥有超过800万活跃商家和数百万骑手的庞大配送网络。2025年年度交易用户数突破8亿。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约5200亿港元（2026年5月，按84港元/股估算） |
| PE(TTM) | 亏损，无法计算（2025年净亏损234亿元） |
| ROE | 负值（2025年亏损） |

| 净利率 | 负（亏损状态） |
| 资产负债率 | 约34% |
| 自由现金流 | 数据待确认 |

### 护城河分析

美团的护城河来自其即时配送网络的规模效应。数百万骑手覆盖全国，配送效率行业领先，商家和用户的多边网络效应强。到店酒旅业务有大众点评的评价数据积累，是另一层壁垒。在外卖领域，尽管面临京东、抖音等新进入者的挑战，美团凭借更成熟的运力体系和更低的亏损水平守住了60%以上的份额。

但2025年的财报暴露了护城河的脆弱面：京东外卖以补贴策略猛攻，迫使美团大幅增加补贴投入，核心本地商业板块从盈利转为亏损69亿元。这说明美团的护城河虽宽，但并非不可撼动，尤其在对手愿意持续烧钱的情况下。到店酒旅也面临抖音本地生活的持续渗透。美团的护城河更像是"运营效率型"而非"结构性"的，需要持续投入维护。

### 管理层简评

王兴是中国互联网行业最受尊敬的CEO之一，连续创业经历（校内网、饭否、美团）展现了极强的战略判断力和执行力。他在"千团大战"中胜出、成功整合大众点评、击退饿了么的经历证明了其竞争能力。王兴提出的"反内卷""做正确的事"理念也体现了长期思维。但当前面临的挑战是：面对京东外卖的进攻，管理层被迫选择大规模补贴应战，导致利润表剧烈恶化。这是否是被动之举还是战略主动，值得深入分析。海外业务Keeta在香港已盈亏平衡，显示出国际化的初步成效。

### 估值判断

美团当前处于亏损状态，传统PE估值失效。按2024年经调整净利润358亿元计算，当前市值对应约15倍PE，并不算贵。但关键问题是2025年巨亏后，盈利何时恢复。如果竞争态势缓和，美团凭借其运营效率应能较快恢复盈利。但如果京东外卖持续烧钱，亏损期可能延长。目前股价较52周高点149.8港元腰斩，反映了市场的悲观预期。对于能承受波动的投资者，这可能是一个被过度悲观定价的机会，但风险也不小。

### 初步评级

- 生意质量：★★★★
- 护城河强度：★★★
- 管理层质量：★★★★★
- 估值吸引力：★★★
- 综合评分：★★★★
- 是否值得深入研究：是
- 一句话总结：中国本地生活绝对龙头，王兴能力一流，但外卖大战致盈利剧烈恶化，需要判断竞争何时缓和才能下手。

## 京东集团（9618.HK）

### 核心业务

京东是中国第二大电商平台（按营收计第一），核心业务是自营电商（3C数码、家电、日百等品类），同时运营第三方商家平台、京东物流、京东健康等。2025年全年营收13091亿元，是中国电商行业唯一万亿级收入体量的公司。京东零售收入11264亿元占主导，服务收入占比21.8%创新高。京东的核心差异化在于自建物流体系——全国覆盖的仓储配送网络保证了"当日达/次日达"的履约体验。员工总数超90万，其中大部分是一线物流人员。2025年京东进入外卖市场，推出京东外卖独立App，是重大战略扩张。

### 关键财务指标

| 指标 | 数值 |
|------|------|
| 市值 | 约3354亿港元（2026年5月，按130港元/股估算） |
| PE(TTM) | 约10倍（Non-GAAP口径） |
| ROE | 数据待确认（利润下降较大） |
| 营收（2025年） | 13091亿元 |
| 营收增长率 | 13% |
| 净利润（2025年） | 196亿元（GAAP）；270亿元（Non-GAAP） |
| 净利润增长率 | -53%（GAAP）；-43%（Non-GAAP） |
| 毛利率 | 15.6% |
| 净利率 | 约2.2%（GAAP） |
| 资产负债率 | 52.1% |
| 自由现金流 | 65亿元（同比下降85%） |

### 护城河分析

京东的护城河主要来自自建物流网络。多年持续投入形成的全国仓配体系是竞争对手难以短时间复制的重资产壁垒，保证了京东在3C数码、家电等标品品类的履约优势。京东的用户认知也很清晰："正品+快速配送"。供应链管理能力是其核心竞争力，京东零售的经营利润率持续提升至4.6%就是效率优化的体现。

弱点同样明显：毛利率仅15.6%，远低于平台型电商（阿里40%+），这是自营模式的固有局限；净利率只有2%出头，盈利能力薄弱。外卖业务是一场豪赌——2025年外卖亏损148亿元，严重拖累利润和现金流。京东在女性用户、下沉市场、内容化电商方面的短板也长期存在。物流虽是护城河，但也是沉重的成本负担。

### 管理层简评

刘强东是京东的灵魂人物，2023年后重新强化对公司的掌控，持股比例升至69.35%（投票权）。刘强东推动了低价策略和外卖业务扩张，展现了强势决策风格。CEO许冉是财务出身，在"降本增效"方面做出了成绩，京东零售利润率持续提升。但管理层面临的质疑是：外卖市场投入是否值得？2025年外卖亏损148亿元，自由现金流从437亿暴跌至65亿，这是否是战略误判还是长期布局？刘强东的集权式管理风格有利于决策效率，但也增加了决策风险。

### 估值判断

当前约10倍PE（Non-GAAP），是港股互联网公司中最便宜的之一。13091亿元的营收体量、强大的物流基础设施、90万员工，对应仅3354亿港元市值，看起来很便宜。但便宜有其道理：利润暴跌、自由现金流枯竭、外卖持续烧钱前景不明。如果剔除外卖业务亏损，京东零售本身的估值确实偏低。高盛曾指出"中国电商价值重估空间巨大"，但重估的前提是盈利能力的恢复。

### 初步评级

- 生意质量：★★★
- 护城河强度：★★★
- 管理层质量：★★★
- 估值吸引力：★★★★
- 综合评分：★★★
- 是否值得深入研究：是（但优先级低于腾讯、美团）
- 一句话总结：万亿营收+自建物流的硬资产价值被低估，但自营电商利润率先天不足，外卖烧钱是最大变量。

## 网易（9999.HK）

### 核心业务

网易是中国第二大游戏公司，也是一家以内容驱动的互联网公司。主营业务包括在线游戏（收入占比约80%）、有道教育、云音乐和创新业务。2025年全年营收1126亿元，其中游戏及相关收入921亿元，网易云音乐78亿元，有道59亿元。网易在中国游戏市场份额约28%，与腾讯（50%）合计近80%，是典型的双寡头格局。网易以自研精品游戏著称，拥有《梦幻西游》《阴阳师》《逆水寒》《永劫无间》等头部产品，是中国武侠/仙侠题材游戏出海最成功的公司。研发投入连续六年超100亿元（2025年177亿元），是游戏行业研发投入最大的公司之一。

按核心运营利润约134亿元计算，当前PE约20倍，并不算便宜。表面上14倍PE有吸引力，但需要剔除大量非经常性投资收益的干扰。携程反垄断调查的不确定性是当前最大的估值压制因素——如果被要求大幅降低佣金，利润将显著受损。81%的毛利率也可能被监管"盯上"。在竞争加剧和监管风险并存的背景下，当前估值谈不上低估。股价从613港元高点回落至420港元左右，部分反映了这些风险。

## 腾讯音乐（1698.HK）

### 核心业务

腾讯音乐（TME）是中国最大的在线音乐平台，旗下拥有QQ音乐、酷狗音乐、酷我音乐三大平台，合计市场份额约69%。核心业务分为在线音乐服务（订阅+广告+数字专辑等）和社交娱乐服务（直播打赏等，占比持续缩小）。2025年全年总收入329亿元，同比增长15.8%，调整后净利润99.2亿元，同比增长22%。在线音乐订阅收入177亿元，付费用户1.274亿。超级会员突破2000万，是提升ARPU的关键举措。腾讯音乐还积极拓展IP衍生业务，包括线下演唱会、专辑销售和粉丝经济。2026年5月宣布收购喜马拉雅，进一步扩展音频版图。

### 护城河分析

腾讯音乐的护城河来自其在音乐版权方面的绝对优势和用户规模。QQ音乐、酷狗、酷我三大平台的曲库资源是行业最全的，69%的市场份额使其在与唱片公司的版权谈判中有强大议价权。付费用户1.274亿形成了稳定的订阅收入基础。腾讯系的流量支持（微信、QQ）也是竞争对手难以获取的渠道优势。

但护城河也在面临新的挑战：字节跳动的汽水音乐正在快速渗透，网易云音乐在社区和个性化推荐方面有差异化优势。更根本的问题是，音乐流媒体的用户ARPU偏低（对比视频流媒体），天花板不高。直播打赏业务持续萎缩，传统社交娱乐收入在下降。腾讯音乐的收购喜马拉雅虽然扩展了版图，但整合风险不容忽视。

### 管理层简评

腾讯音乐的管理层受腾讯集团的强影响，战略方向与腾讯整体协同。管理层在订阅化转型方面执行力强，将付费率从低个位数提升至行业领先水平，超级会员策略有效提升了ARPU。IP衍生业务（演唱会、周边）的布局显示了管理层在多元变现上的探索。2025年利润增长22%的同时保持15.8%的收入增长，说明运营效率在持续提升。但管理层面临的挑战是如何在汽水音乐的竞争压力下保持增长势头，以及如何整合喜马拉雅。

### 估值判断

股价从2025年8月高点103港元暴跌至35港元（跌幅66%），当前PE仅约10倍，在港股互联网板块中属于极低水平。市场大跌的原因包括：2026年Q1增速放缓（收入仅增7.3%）、汽水音乐竞争加剧的担忧、喜马拉雅收购的不确定性。但从基本面看，329亿收入、99亿调整后利润、44%毛利率、69%市场份额的公司仅值1400亿港元，存在明显的错杀可能。如果增长能保持在10%以上，10倍PE就是低估。风险在于增长进一步放缓和竞争格局恶化。
