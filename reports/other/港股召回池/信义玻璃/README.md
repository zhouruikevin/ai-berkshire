## 研究框架

本研究采用四维度分析框架：

1. **商业模式分析** — 生意本质、护城河、定价权
2. **财务估值分析** — 盈利质量、资本配置、安全边际
3. **行业竞争分析** — 产业格局、竞争壁垒、变化趋势
4. **风险与管理层评估** — 核心风险、治理结构、资本配置能力

## 研究背景

信义玻璃是中国最大的综合玻璃制造商之一，浮法玻璃日熔量近30000吨，位居全球前列。当前处于浮法玻璃行业下行周期尾部（始于2023Q4），汽车玻璃业务逆势增长，已成为利润核心支柱。公司正通过马来西亚、印尼、沙特等海外产能布局实现国际化突破。
