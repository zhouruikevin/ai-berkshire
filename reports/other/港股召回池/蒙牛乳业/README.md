## 研究背景

蒙牛是中国乳制品行业双寡头之一（另一家为伊利股份），2025年实现收入822.4亿元。公司正处于管理层换代后的战略调整期：前任留下的并购减值包袱（贝拉米43.3亿商誉减值、现代牧业联营亏损等）正在出清，新任总裁高飞推行"一体两翼"战略，从规模扩张转向利润质量提升。当前PE约15倍，市场等待2026年利润拐点确认。

## 核心矛盾

1. **利润拐点何时确认？** 2025年经营现金流创历史新高（87.5亿），但收入同比下滑7.3%，减值出清尚未完成
2. **与伊利差距是否可逆？** 营收差距已扩大至334亿，市值差距近3倍
3. **原奶周期能否助力？** 行业预计2026年供需再平衡，对下游有利
4. **管理层能力是否匹配？** 高飞"老蒙牛"出身，但尚未证明扭转能力
