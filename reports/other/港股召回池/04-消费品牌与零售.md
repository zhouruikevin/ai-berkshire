## 农夫山泉（9633.HK）

### 核心业务
农夫山泉是中国最大的包装饮用水和饮料企业之一，主营业务涵盖包装饮用水、茶饮料、功能饮料、果汁饮料等品类。公司拥有十二大优质水源地，商业模式核心是"天然水+饮料双轮驱动"。2025年茶饮料板块营收突破215.96亿元，同比增长29%，占总营收比重41.1%，首次超越饮用水成为第一大收入来源；饮用水业务营收187.09亿元，同比增长17.3%。公司在包装饮用水市场连续多年市占率第一，东方树叶已成为无糖茶赛道的绝对龙头。收入构成从"水为主"转向"茶水双驱"，产品结构持续优化。

### 护城河分析
农夫山泉的护城河由三层构成。第一层是水源地壁垒：公司在全国锁定十二大稀缺天然水源地，后来者几乎不可能复制这种先发布局，这是真正的资源型壁垒。第二层是品牌和渠道壁垒：从"农夫山泉有点甜"到"大自然的搬运工"，公司长期积累的品牌心智极强，线下分销网络覆盖数百万终端，渠道下沉深度远超同行。第三层是产品创新能力：东方树叶在无糖茶赛道的提前布局堪称教科书级别，2011年推出时被嘲笑为"最难喝的饮料"，十余年后成为行业领军。

反面来看，饮用水赛道的同质化竞争依然激烈，怡宝、娃哈哈等对手持续施压；茶饮料赛道涌入大量竞争者，东方树叶的先发优势能否长期维持需要观察。此外，公司高度依赖创始人钟睒睒的个人判断力，这是优势也是风险——66岁的钟睒睒尚无明确接班安排。

### 管理层简评
创始人钟睒睒持股约87.5%，对公司拥有绝对控制权。他的商业直觉极强，从包装水到东方树叶的每一次关键决策都踩准了行业节奏，战略眼光和执行力在消费品行业属于顶级。但钟睒睒风格偏"独裁"，公司治理高度依赖个人，传承风险是主要隐患。目前其子已进入养生堂管理层，但能否继承父辈的商业天赋仍是未知数。管理层整体执行力强，费用管控出色，销售费用率从22.6%降至18.6%。

### 估值判断
PE(TTM)约28倍，对应44%的ROE和30%以上的利润增速，估值不算离谱但谈不上便宜。农夫山泉是A股/港股稀缺的消费品平台型公司，享受一定的估值溢价是合理的。但当前价格已经反映了茶饮料高增长的预期，未来增速若放缓，估值可能面临压缩。如果PE回到25倍以下，安全边际会更充足。

### 初步评级
- 生意质量：★★★★★
- 护城河强度：★★★★☆
- 管理层质量：★★★★☆
- 估值吸引力：★★★☆☆
- 综合评分：★★★★☆
- 是否值得深入研究：是
- 一句话总结：消费品领域罕见的高ROE+高增长组合，生意质量一流，但估值需要耐心等待更好的价格。

## 泡泡玛特（9992.HK）

### 核心业务
泡泡玛特是全球领先的潮流文化娱乐公司，以IP运营为核心，通过设计、生产和销售潮流玩具及衍生品变现。公司建立了从IP孵化、产品开发到全渠道零售的完整产业链。2025年营收371.2亿元，同比增长184.7%，其中THE MONSTERS（含LABUBU）全年收入141.6亿元，占总营收38.1%。海外及港澳台业务营收50.7亿元（上半年），同比增长375.2%。公司拥有MOLLY、DIMOO、SKULLPANDA、THE MONSTERS等多个头部IP矩阵，全球门店超500家，在超过80个国家和地区销售。

### 护城河分析
泡泡玛特的护城河由IP运营能力、供应链管理和全球渠道网络三层组成。IP运营是核心：超200人设计团队，专属IP经理全流程负责，与头部设计师（如LABUBU创作者龙家升）建立深度绑定，竞争对手难以复制这种"艺术+商业"的平衡能力。供应链端，公司与核心代工厂深度协作，建立滚动下单机制，品控能力强。全球渠道网络快速扩张，品牌势能在东南亚等市场已形成文化穿透力。

但风险同样突出：LABUBU单一IP占营收38%，依赖度过高，一旦热度衰退，业绩波动可能很大。潮玩行业天然面临"IP生命周期"问题，历史上没有哪个潮玩IP能火超过5年不衰。管理层虽然承诺2026年20%+增长，但从184%骤降至20%，市场预期差可能引发剧烈波动。此外，潮玩消费具有可选属性，经济下行期需求弹性大。

### 管理层简评
创始人王宁是泡泡玛特的灵魂人物，从街边小店到千亿市值，展现了极强的商业嗅觉和战略定力。他对IP生命周期管理的理解深刻——"IP企业最重要的就是持续运营IP"——这种长期主义思维是泡泡玛特能跑出来的关键。在LABUBU爆火后，王宁选择主动降速修底盘而非激进扩张，显示出难得的克制。团队执行力强，海外拓展速度远超预期。

### 估值判断
当前PE(TTM)约17倍，看起来不贵，但这是建立在2025年爆发式增长的高基数上。如果2026年增速回落至20%，对应的PEG约0.85，合理偏低。关键问题是2025年的业绩是否可持续——LABUBU的全球热度能否维持、新IP能否接力、海外市场能否持续高增。如果能证明增长的可持续性，当前估值有吸引力；如果LABUBU热度消退而新IP未能补位，17倍PE并不算安全。

### 初步评级
- 生意质量：★★★★☆
- 护城河强度：★★★☆☆
- 管理层质量：★★★★☆
- 估值吸引力：★★★★☆
- 综合评分：★★★★☆
- 是否值得深入研究：是
- 一句话总结：IP运营能力出色、增长爆发力惊人，但单一IP依赖和潮玩行业的周期性是核心风险，需要持续跟踪IP矩阵的多元化进展。

## 安踏体育（2020.HK）

### 核心业务
安踏体育是中国最大的运动鞋服集团，通过"单聚焦、多品牌、全渠道"战略构建了覆盖大众到高端的品牌矩阵。主品牌安踏定位大众运动（2025年营收347.54亿元，占比43.3%），FILA定位中高端时尚运动（284.69亿元），其他品牌包括始祖鸟（Arc'teryx）、迪桑特（Descente）、可隆（Kolon）等高端户外品牌。2025年集团总营收802.19亿元，同比增长13.3%，首次突破800亿大关，已超越耐克中国成为中国运动鞋服市场第一。集团采用DTC（直营零售）和批发并行的渠道模式。

### 关键财务指标

|------|------|
| 市值 | 约2415亿港元 |
| PE(TTM) | 约14倍 |
| ROE | 约25.4% |
| 营收（最新年度） | 802.19亿元（2025年） |

*数据来源：公司2025年年报、券商研报。净利润下降主要因亚玛芬体育相关费用影响*

### 护城河分析
安踏的护城河在于多品牌协同管理能力，这是竞争对手最难复制的核心壁垒。收购FILA并成功复兴是安踏证明自己的标志性案例——将一个衰落的意大利品牌在中国做到年营收近300亿元，堪称教科书。始祖鸟、迪桑特等高端品牌的成功运营进一步验证了这种能力的可复制性。从"大众-中高端-高端户外"的金字塔品牌矩阵，到覆盖全价格带的消费者触达，安踏的多品牌战略构建了宽阔的护城河。

但挑战也很明显：FILA增长已现疲态，2025年营收增速放缓且毛利率下滑；始祖鸟等新品牌虽然增速快但体量尚小；主品牌安踏面临李宁、特步等对手的持续竞争。此外，多品牌管理的复杂度随规模扩大而增加，整合风险不容忽视。

### 管理层简评
创始人丁世忠是中国体育用品行业最具战略远见的企业家之一，从晋江小厂到全球第三大运动品牌集团，每一步都踩对了节奏。他提出的"三大核心能力"——多品牌协同管理、多品牌零售运营、全球化运营整合——已经被业绩证明。管理层整体执行力强，对FILA管理团队给予充分自主权的做法体现了格局。隐忧在于丁氏家族控股，公司治理的职业化程度仍需提升。

### 估值判断
PE(TTM)约14倍，对应25%的ROE和800亿+的营收体量，估值在港股运动品牌中偏低。2025年净利润下降12.9%是估值承压的主因，但剔除亚玛芬相关影响后核心利润仍在增长。如果亚玛芬（始祖鸟母公司）2026年IPO成功，安踏的持股价值将得到重估。当前估值已经反映了市场对FILA放缓的担忧，14倍PE对于一个800亿营收、25%ROE的消费巨头而言具有一定吸引力。

### 初步评级
- 生意质量：★★★★☆
- 护城河强度：★★★★☆
- 管理层质量：★★★★★
- 估值吸引力：★★★★☆
- 综合评分：★★★★☆
- 是否值得深入研究：是
- 一句话总结：中国运动品牌的绝对龙头，多品牌运营能力是真正的核心竞争力，当前估值合理偏低，值得重点跟踪。

## 李宁（2331.HK）

### 核心业务
李宁是中国领先的运动品牌公司，由"体操王子"李宁创立，以"中国李宁"为核心IP，主营运动鞋、运动服装及器材。2025年营收295.98亿元，同比增长3.2%，创历史新高。鞋类收入146.51亿元（占比49.5%），服装收入123.27亿元（占比41.6%），器材及配件26.21亿元。渠道结构上，批发137.73亿元、电商87.43亿元、直营66.55亿元。公司定位"专业运动+运动时尚"，在国潮赛道拥有独特品牌心智。

### 关键财务指标
| 指标 | 数值 |
|------|------|
| 市值 | 约554亿港元 |
| PE(TTM) | 约17倍 |
| ROE | 约12.2%（2025年） |
| 营收（最新年度） | 295.98亿元（2025年） |
| 营收增长率 | 3.2% |
| 净利润（最新年度） | 29.36亿元（归母） |
| 净利润增长率 | -2.6% |
| 毛利率 | 49.0% |
| 净利率 | 9.9% |
| 资产负债率 | 26.1% |
| 自由现金流 | 经营现金流48.52亿元 |

*数据来源：公司2025年年报、券商研报*

### 护城河分析
李宁的护城河集中在品牌力这一个维度——"中国李宁"四个字本身就是一个超级IP。2018年纽约和巴黎时装周走秀开创了"国潮"概念，将中国传统文化元素与现代运动时尚结合，形成了差异化的品牌定位。这种"民族品牌+运动基因+国潮时尚"的三位一体定位，短期内竞争对手难以模仿。

但李宁的护城河宽度远不如安踏。首先，品牌力过度依赖"国潮"热度，一旦潮流退去，品牌溢价可能随之缩水。其次，李宁是单品牌运营，抗风险能力弱于安踏的多品牌矩阵。第三，毛利率49%远低于安踏的62%，说明品牌溢价能力和渠道效率都有差距。第四，营收增速仅3.2%且净利润负增长，说明公司正处于增长瓶颈期。渠道管理、库存控制等运营层面与安踏差距明显。

### 管理层简评
李宁本人2015年回归后挽救了公司，引入前优衣库高管钱炜担任联席CEO推动改革，组织架构从职能导向变为事业部制，效率有所提升。但管理层在战略执行上缺乏安踏的稳定性——过去几年在高端化和大众化之间摇摆，品牌定位一度模糊。2026年奥运周期可能带来催化剂，但管理层需要证明能把品牌势能转化为持续增长。目前来看管理层改革方向正确但执行力有待提高。

### 估值判断
PE约17倍，对应3%的营收增速和12%的ROE，估值谈不上便宜。与安踏相比，李宁的ROE低一半、营收体量不到安踏的四成、增速更慢，但PE却更高——说明市场对李宁的品牌溢价给了过高定价，或者对2026年奥运催化剂有预期。如果2026年能实现管理层指引的高单位数增长，当前估值可接受；否则有下行风险。

### 初步评级
- 生意质量：★★★☆☆
- 护城河强度：★★★☆☆
- 管理层质量：★★★☆☆
- 估值吸引力：★★★☆☆
- 综合评分：★★★☆☆
- 是否值得深入研究：否（优先关注安踏）
- 一句话总结：品牌基因独特但运营效率与安踏差距明显，增长乏力且估值无明显安全边际，不具备优先深入研究的性价比。

## 海底捞（6862.HK）

### 核心业务
海底捞是中国最大的火锅连锁餐饮企业，以极致服务著称。2025年全年营收432.25亿元，同比增长1.1%。公司在全球运营超过1300家门店，主要收入来自餐厅经营。近年启动"红石榴计划"推进多品牌战略，孵化了小嗨火锅、苗师兄鲜炒鸡、焰请烤肉等子品牌。其他餐厅收入大幅增长，显示多品牌布局初见成效。火锅行业同质化严重，海底捞的差异化在于服务体验和品牌认知度。

| 市值 | 约830-900亿港元（估计） |
| PE(TTM) | 约20倍（估计） |
| ROE | 约35-40%（2025年，估计基于2024年42%趋势下滑） |
| 营收（最新年度） | 432.25亿元（2025年） |
| 营收增长率 | 1.1% |
| 净利润（最新年度） | 40.42亿元 |

### 护城河分析
海底捞的核心护城河并非火锅本身（火锅的技术壁垒几乎为零），而是其独创的"人力资源管理体系"和由此衍生的极致服务文化。"双手改变命运"的价值观让员工拥有清晰的晋升通道，激发了基层员工的主动性和创造力，这套师徒制+计件制的管理模式是海底捞真正难以复制的壁垒。品牌认知度极高，"海底捞"三个字已经成为火锅品类的代名词。

但火锅行业的结构性问题不可忽视：进入门槛低、同质化严重、翻台率趋势性下降。海底捞2025年营收仅增1.1%、净利润下降14%，说明在门店数量接近天花板后，增长引擎已经减弱。多品牌战略尚在早期，能否复制海底捞的成功模式存疑。此外，餐饮行业受宏观经济和消费信心影响大，抗周期能力有限。

## 申洲国际（2313.HK）

### 核心业务
申洲国际是全球最大的纵向一体化针织服装制造商，核心客户包括耐克、阿迪达斯、优衣库、彪马四大国际品牌（合计占收入约80%）。公司业务覆盖面料织造、染整、印绣到成衣裁剪缝纫的全产业链，具备从面料接单到成衣交货最快10天的极速响应能力。2025年营收309.9亿元，同比增长8.1%；净利润58.3亿元，同比下降6.7%。在越南、柬埔寨等地建有海外产能基地，有效应对贸易摩擦。与客户共同研发的功能性面料（如优衣库Airism、耐克Flyknit）体现了ODM而非简单OEM的定位。

### 护城河分析
申洲国际的护城河极其深厚，由"纵向一体化制造能力+头部客户深度绑定+面料研发能力"三层构成。纵向一体化是核心：将面料到成衣的全工序整合，实现了60天以内交货的订单占四成以上，这种效率同行难以企及。与耐克、优衣库等客户20年以上的合作关系构成了天然壁垒——客户转换供应商的切换成本极高。面料共同研发能力（Airism、TechFleece、Flyknit等）使申洲从代工厂升级为创新伙伴，附加值远高于普通OEM。

挑战在于：毛利率仅26.3%且有下降趋势，制造业的利润天花板明显；客户集中度过高（四大客户占80%），任何一个大客户的订单波动都会显著影响业绩；劳动力成本持续上升，海外建厂虽然对冲了部分压力但管理难度增大。此外，全球贸易摩擦和关税政策是制造业出口商永远的不确定性来源。

### 管理层简评
马建荣家族控股（约47.3%），马建荣拥有超过40年纺织行业经验，核心高管平均任职超15年，管理团队稳定性极强。高管合计持股达53.6%，形成高度利益共同体——管理层和股东的利益完全一致，这是难得的治理结构。马建荣低调务实，长期专注制造主业，不搞多元化，这种"做一件事做到极致"的风格与巴菲特喜欢的企业家特质高度吻合。

### 估值判断
PE约15倍，对应稳定的ROE（15-18%）和保守的增长（8%营收增速），估值合理。申洲国际是典型的"稳健优质资产"——不会让你赚到暴利，但大概率不会让你亏钱。2026年预测利润恢复增长（59.4亿元，+1.9%），2027年增速提升至13.4%。如果毛利率企稳回升、客户订单恢复增长，15倍PE的估值具有一定安全边际。适合追求确定性的保守型投资者。
