# 华润电力（0836.HK）：财务与估值分析

> 分析维度：盈利质量、资产负债、现金流、估值与安全边际
> 研究日期：2026年5月16日

---

## 一、盈利能力分析

### 1.1 收入与利润趋势

华润电力近三年的财务表现呈现"先扬后稳"的特征：

| 指标 | 2023年 | 2024年 | 2025年 |
|------|--------|--------|--------|
| 营业额（亿港元） | 1,033 | 1,052.8 | 1,020.1 |
| 归母净利润（亿港元） | 110 | 143.9 | 145.2 |
| 核心业务利润（亿港元） | — | 138.7 | 152.4 |
| 火电核心利润（亿港元） | 36 | 46.4 | 76.4 |
| 新能源核心利润（亿港元） | 97 | 92.3 | 76.0 |
| 每股盈利（港元） | 2.11 | 2.77 | 2.80 |

几个关键观察：

**正面信号**：
- 核心业务利润增长9.9%至152.4亿港元，说明经营质量在提升
- 火电盈利大幅修复（+64.7%），煤价下行红利充分体现
- 整体盈利保持了稳定性，没有出现大幅波动

**负面信号**：
- 营业额同比下降3.1%，上网电价下行（综合均价降幅7.9%）抵消了发电量增长（+7.0%）
- 新能源利润下滑17.6%，这是一个值得警惕的信号——装机在增长，利润在萎缩
- 归母净利润仅增长0.9%，扣除火电的周期性修复后，增长质量堪忧

### 1.2 盈利能力指标

| 指标 | 2023年 | 2024年 | 2025年（估） |
|------|--------|--------|-------------|
| ROE | 13.2% | ~14% | ~12% |
| EBITDA（亿港元） | — | 389.6 | ~400 |
| 净利率 | 10.7% | 13.7% | 14.2% |
| 毛利率（新能源） | — | ~58% | ~56% |

ROE维持在12-14%的水平，在公用事业板块属于中上，但远不及优秀消费或科技企业的20%+。问题在于，ROE的提升很大程度上依赖于杠杆——67%以上的资产负债率意味着股东权益只占三成，放大了权益回报。如果用"真实资产回报率"（ROA）来看，画面远没有那么好看。

新能源业务近60%的毛利率看起来很漂亮，但要注意这是"会计毛利率"——新能源的成本主要在折旧而非变动成本，高毛利率并不代表高现金回报。

### 1.3 利润的"含金量"

2025年有一个重要的会计调整影响：新能源补贴相关的一次性会计估计调整拖累了新能源利润。剔除这一因素后，新能源的经营利润下滑幅度会小一些，但电价下行和弃风弃光的影响是实实在在的。

火电利润的大幅增长（+64.7%）更多是周期性的——2022-2023年煤价高位时火电利润被压缩，2024-2025年煤价回落后利润弹性释放。这种"均值回归"式的增长不应被外推。

## 二、资产负债表分析

### 2.1 资产结构

华润电力是典型的重资产企业。截至2023年末（最近完整披露数据），总资产约3,200亿港元，其中固定资产和在建工程占据绝大部分。2025年中期总资产已增至3,468亿港元。

资产端的核心关注点：

**新能源资产质量**：2025年上半年，新能源资本开支约158亿港元，全年预计420亿港元。这些资产能否在电价持续下行的环境中产生足够回报？如果未来电价进一步下降10-15%，部分项目的IRR可能跌破资金成本。

**可再生能源补贴应收款**：这是电力行业的一个"暗雷"。虽然华润的补贴欠款数据未单独披露，但整个行业的补贴拖欠问题由来已久，华润新能源IPO审核中也被问及此问题。补贴应收款如果出现减值，将直接冲击利润和资产。

### 2.2 负债结构

| 指标 | 数据 |
|------|------|
| 总负债 | ~2,178亿港元（2023年末） |
| 资产负债率 | ~67.4% |
| 归母净资产 | ~850亿港元 |

67%以上的资产负债率在电力行业属于正常水平，但绝对水平不低。华润电力每年568亿港元的资本开支，大部分依赖债务融资，杠杆率有进一步上升的压力。

**积极因素**：央企AAA信用评级，融资成本远低于行业平均。这是华润电力相对于民营新能源企业的核心竞争优势之一。

**风险因素**：如果利率环境变化或信用市场收紧，高杠杆将放大财务风险。不过，考虑到当前中国处于降息周期，短期内这个风险可控。

### 2.3 资本效率

华润电力的"资本回收周期"较长。以新能源项目为例：假设一个风电项目投资10亿，年发电收入1.5亿，扣除运维和利息后净收入约0.5亿，简单回收期约20年。这意味着公司需要不断投入新资本才能维持增长，而旧项目的回报在电价下行环境中可能不断被侵蚀。

这种"跑步机效应"是重资产行业的通病：你不能停下来，因为折旧在消耗存量资产价值；但你越跑越快，资本需求越来越大。

## 三、现金流分析

### 3.1 经营现金流

华润电力的经营现金流相对稳健，EBITDA约400亿港元，经营现金流大致在250-300亿港元区间（扣除营运资金变动和税费后）。

### 3.2 自由现金流

这是巴菲特最关注的指标——自由现金流 = 经营现金流 - 资本开支。

以2025年为例：
- 经营现金流：约250-300亿港元
- 资本开支：约568亿港元
- **自由现金流：约-270至-320亿港元**

没错，华润电力目前是**自由现金流为负**的。公司赚的钱不够投的，缺口靠借钱来填补。这是"成长型公用事业"的典型困境——为了未来的装机增长，当前在大量烧钱。

**这对估值意味着什么？**

如果按DCF（现金流折现）模型来估值，华润电力在未来5-8年内的自由现金流可能持续为负或极低。公司的价值更多依赖于"未来某一天资本开支放缓后释放出来的现金流"。这使得DCF估值对假设极度敏感，不确定性很大。

### 3.3 股息可持续性

2025年每股派息1.127港元，派息率40.2%，股息率5.7%。在自由现金流为负的情况下，股息本质上是通过"借钱分红"来实现的。这并非不可持续——只要公司能持续融资，就能持续分红——但它依赖于资本市场的持续支持和信用评级的维持。

## 四、估值分析

### 4.1 当前估值水平

| 指标 | 数据 |
|------|------|
| 股价 | 18.26港元 |
| 市值 | 941亿港元 |
| PE（TTM） | ~7.1倍 |
| PB | ~1.0倍 |
| 股息率 | 5.7% |
| EV/EBITDA | ~6倍 |

### 4.2 估值是否便宜？

**看起来便宜的理由**：
- PE 7倍在全球公用事业板块中属于偏低水平
- 股息率5.7%显著高于港股平均和国债收益率
- PB约1倍，接近净资产
- 分析师目标价均值21.3港元，隐含17%上涨空间

**可能不便宜的理由**：
- 自由现金流为负，PE的"分母"（利润）并不完全属于股东
- 火电利润中有较大周期性成分，如果煤价回升，利润可能回落
- 新能源利润增速在放缓，电价下行趋势尚未见底
- 中国电力行业整体PE中枢偏低，7倍PE在同行中并不算特别便宜

### 4.3 华润新能源分拆的估值影响

华润新能源A股IPO募资245亿元，已获证监会注册。分拆上市对母公司华润电力有几层影响：

**正面**：
- A股给新能源资产的估值通常远高于港股，分拆可以释放被港股低估的价值
- 245亿募资将直接降低华润新能源的资产负债率，改善财务结构
- 母公司仍持有控股权，可以并表

**风险**：
- 分拆后母公司剩余资产主要是火电，可能面临"好资产被拿走"的估值折价
- 新能源业务独立后与火电的协同效应可能削弱
- A股IPO市场变数仍多，发行定价和节奏可能不及预期

### 4.4 安全边际评估

按巴菲特的"安全边际"框架：

**乐观情景**（30%概率）：煤价持续低位、电价企稳、新能源消纳改善，2026年利润增长15-20%，对应PE 6倍，加上分拆催化，股价有30-40%上涨空间。

**基准情景**（50%概率）：煤价小幅回升、电价温和下行、新能源利润持平，2026年利润持平或微增，股价在当前水平附近波动，赚取5-6%的股息收益。

**悲观情景**（20%概率）：煤价大幅反弹、电价加速下跌、弃风弃光恶化，2026年利润下滑15-20%，股价可能回落至15-16港元区间。

综合来看，当前估值提供了"有限的安全边际"——不是特别便宜，但也不贵。下行空间约10-15%，上行空间约20-30%，风险收益比尚可。

## 六、小结

华润电力的财务画面是一个"矛盾体"：利润表看起来还不错（145亿利润，12%的ROE），但现金流量表暴露了问题（自由现金流为负，大量资本开支）。资产负债表处于可控但不算舒适的状态（67%负债率）。

从巴菲特的视角看，华润电力不是一家"优秀"的企业——它赚的钱几乎全部（甚至更多）要重新投入到生意中去，股东拿到的真实回报远低于账面利润所暗示的。但在7倍PE和5.7%股息率的估值下，它可能是一个"合理价格买合理企业"的机会——前提是投资者对电力行业的长期需求增长和华润的运营效率有足够信心。
