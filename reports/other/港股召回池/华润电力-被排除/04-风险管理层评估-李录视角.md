# 华润电力（0836.HK）：风险与管理层评估

> 分析维度：下行风险、管理层能力、治理结构、央企特殊性
> 研究日期：2026年5月16日

---

## 一、风险全景扫描

李录的投资框架强调"先看风险，再看机会"。对华润电力而言，风险可以分为三个层次：致命风险、重大风险和可控风险。

### 1.1 致命风险：有没有能让公司"死掉"的因素？

**答案：没有。**

华润电力作为央企控股的大型电力企业，几乎不存在破产风险。理由如下：
- 电力是国家安全和经济运行的基础设施，不可能被允许倒闭
- 华润集团（母公司）总资产超2万亿港元，有充足的资源支持
- 央企AAA信用评级，融资渠道畅通
- 电力资产有实物价值托底

但"不会死"不等于"投资者不会亏钱"。公司可以不倒闭，股价却可以长期低迷。

### 1.2 重大风险：可能导致利润大幅下滑的因素

**风险一：绿电电价持续下行 [影响程度：★★★★★]**

这是当前华润电力面临的最核心风险。数据触目惊心：
- 风电均价：从0.45元 -> 0.44元 -> 0.35元/千瓦时（两年降幅22%）
- 光伏均价：从0.37元 -> 0.32元 -> 0.28元/千瓦时（两年降幅24%）

136号文要求新能源全面参与市场化交易后，绿电定价的"保护伞"正式消失。在新能源装机增速远超需求增速的背景下，电价下行的压力是结构性的，不是周期性的。

华润电力2025年新能源核心利润已经同比下滑17.6%。如果电价再跌15-20%，新能源业务的利润可能腰斩。

**风险二：弃风弃光问题加剧 [影响程度：★★★★☆]**

华润新能源的弃风弃光率数据令人担忧：
- 弃风率：2.80% -> 4.16% -> 5.26%（逐年上升）
- 弃光率：1.52% -> 6.59% -> 12.73%（急剧恶化）

弃光率在两年内从1.52%飙升至12.73%，意味着每100度光伏发电有近13度被浪费。这直接降低了光伏项目的实际收益率。

消纳问题的根源在于电网建设滞后于新能源装机。解决这个问题需要巨额电网投资和储能配置，这不是华润电力单方面能解决的。

**风险三：煤价反弹风险 [影响程度：★★★☆☆]**

2025年火电利润大增64.7%，主要受益于煤价下行。2026年行业预判动力煤价格中枢可能"小幅上移"。火电业务的利润对煤价极度敏感——煤价每上涨50元/吨，可能吃掉十几亿的利润。

不过，容量电价政策的调整（回收比例提至50%）为火电利润提供了一定缓冲。即使煤价回升，火电也不太可能重演2021-2022年的大幅亏损。

**风险四：新能源补贴减值 [影响程度：★★★☆☆]**

可再生能源补贴欠款是整个行业的历史遗留问题。华润新能源IPO审核中，监管部门对此进行了重点问询。如果部分项目最终无法纳入补贴清单或补贴政策进一步变化，公司面临确认部分收入减值或退回补贴款的风险。

具体金额未完全披露，但考虑到华润新能源的规模，潜在影响可能达数十亿级别。

**风险五：资本开支失控 [影响程度：★★★☆☆]**

2025年全年资本开支568亿港元，其中420亿用于新能源。在电价下行、消纳困难的环境下，大规模资本开支是否理性？

这涉及到一个根本问题：华润是在"理性投资"还是在"完成任务"？央企的装机指标有一定的政治任务属性，管理层可能在明知回报率下降的情况下仍需推进项目。

### 1.3 可控风险

**利率风险**：高杠杆企业天然对利率敏感。但当前中国处于降息周期，短期利率风险偏低。

**政策变动风险**：电力行业政策频繁调整，但总体方向（市场化+清洁化）是明确的。政策风险更多是"节奏"和"力度"的问题，而非方向性逆转。

**汇率风险**：港元计价资产，人民币收入，存在一定汇率错配。但华润的大部分资产和负债都在境内，汇率影响有限。

## 二、管理层评估

### 2.1 管理层架构

华润电力的管理架构为"集团控股 + 职业经理人"模式：
- **控股方**：华润（集团）有限公司，国务院国资委直管央企
- **集团董事长**：王祥明
- **华润电力总裁**：王波（2024年12月获委任）

王波作为新任总裁，在2025年3月工作会议上提出了"十五五"规划方向，聚焦清洁高效发电、综合能源服务、能源科技创新三大板块。

### 2.2 央企管理层的特殊性

从李录的视角看，管理层评估需要特别关注"管理层与股东利益是否一致"。央企管理层有其独特性：

**优势**：
- **职业化程度高**：央企高管通常有丰富的行业经验和专业背景
- **长期视角**：不像民企创始人可能有套现冲动，央企管理层考核注重长期可持续发展
- **资源调动能力强**：能够通过集团和政府关系获取优质资源

**劣势**：
- **激励不足**：央企高管的薪酬与市场化企业相比偏低，股权激励有限，缺乏"做大蛋糕"的强烈动力
- **轮岗制度**：央企高管通常3-5年轮岗一次，"铁打的公司流水的管理层"导致战略的连续性和深度执行存在不确定性
- **多目标约束**：管理层需要同时满足国资委考核（利润、安全、社会责任）、政治要求（装机指标、碳达峰任务）和市场预期（股价、分红），多目标之间可能冲突
- **决策效率**：重大投资决策需要层层审批，反应速度不及民企

### 2.3 管理层的战略判断力

华润电力过去几年的几个关键战略决策值得复盘：

**做对的事情**：
1. **提前布局新能源**：2019年就开始加速风光布局，先发优势明显，抢占了较多优质风光资源
2. **分拆新能源上市**：在A股市场对新能源资产高估值的窗口期推进IPO，时机选择不错
3. **保留火电底仓**：没有像某些企业那样"all in"新能源，保持了火电+新能源的均衡配置

**值得商榷的**：
1. **装机速度是否过快**：在弃风弃光率快速上升的背景下，2025年仍规划新增10GW风光装机，是否存在"为增长而增长"的倾向？
2. **资本开支管理**：568亿年度资本开支对应145亿利润，资本投入产出比偏低
3. **电价风险对冲**：面对绿电电价持续下行，公司似乎缺乏有效的风险对冲手段

### 2.4 资本配置能力

这是李录最关注的管理层能力之一——公司赚了钱后怎么花？

华润电力的资本配置优先级大致是：
1. **新能源投资**（约420亿/年）：最大的资金去向
2. **分红**（约58亿港元）：派息率40.2%
3. **火电运维和升级**（约100-150亿/年）
4. **债务偿还**：净负债实际在增加

问题在于：在新能源回报率下行的趋势中，将70%以上的资本开支投向新能源是否是最优选择？是否应该提高分红比例、回购股份或减缓投资节奏？

央企的回答往往是"这是国家战略，必须投"。但对小股东而言，"国家利益"和"股东利益"不总是一回事。

## 三、治理结构评估

### 3.1 股权结构

华润（集团）有限公司持有华润电力约62.95%的股权，处于绝对控股地位。这种高度集中的股权结构有利有弊：

**利**：大股东利益与公司利益高度绑定，不存在"野蛮人"入侵的风险，战略方向明确。

**弊**：小股东话语权有限，大股东可能通过关联交易、资源调配等方式影响上市公司利益。

### 3.2 关联交易风险

作为华润集团旗下的上市平台，华润电力与集团之间存在大量关联交易（煤炭采购、资产注入、管理费等）。需要持续关注关联交易是否公允。

### 3.3 信息披露质量

华润电力的信息披露总体规范，作为港股上市公司，需遵守港交所的披露规则。但与A股公司相比，港股在季度数据披露频率和细度上有所不足。

### 3.4 华润新能源IPO相关风险

华润新能源IPO审核中，深交所提出了两轮问询，关注点包括：
- 200+条行政处罚记录
- 可再生能源补贴确认的合规性
- 关联交易的公允性
- 盈利可持续性（2025年净利润同比下滑23.27%至61.02亿元）

这些问询揭示了一些此前在港股财报中不够透明的风险点，值得投资者重视。

## 四、"买股票就是买人"——综合评判

李录强调"买股票就是买人"，但对央企而言，"人"的因素既重要又有限：

**重要的是**：管理层的战略方向、执行效率和资本配置能力
**有限的是**：个人能力在央企体制中的发挥空间受限于制度框架

华润电力的管理层在战略方向上没有犯大错——提前布局新能源、推进分拆上市、保持均衡配置。但在"为股东创造价值"方面，央企管理层天然缺乏足够的激励去最大化股东回报。

对于华润电力这样的央企，投资者不应期望管理层有"创业者"般的激情和魄力。合理的期望是：管理层能够（1）避免重大战略失误，（2）保持运营效率在行业前列，（3）维持合理的分红水平。从目前的表现看，这三点大体满足。

## 六、小结

从李录的投资哲学出发，投资华润电力的核心问题是：**在这个价格下，风险是否被充分补偿？**

**风险端**：电价下行、消纳困难、资本开支过大是三个切实的威胁，可能导致未来2-3年利润增速低于预期甚至下滑。央企管理层的激励机制也意味着"为股东利益最大化"不是第一优先级。

**补偿端**：7倍PE和5.7%股息率提供了一定的缓冲。更重要的是，华润电力不会"死"——最坏的情况是利润下滑20-30%，对应PE也只是从7倍上升到9-10倍，仍在合理范围内。

**结论**：华润电力是一个"风险有限、回报也有限"的标的。它适合追求稳定收益（股息+温和增长）的投资者，但不太适合追求高成长、高弹性的投资者。关键的决策变量是——你对中国电力行业电价走势的判断，以及你对"央企治理折价"的容忍程度。
