## 研究背景

中芯国际是中国大陆规模最大、技术最先进的集成电路晶圆代工企业，2025年销售收入93.27亿美元（673亿元人民币），全球晶圆代工市场份额约4.8%，位居全球第二（仅次于台积电）。公司处于美国实体清单制裁之下，同时受益于中国半导体国产替代大潮。当前估值处于历史高位，市场给予大量"国产替代溢价"。

## 核心矛盾

1. **估值与盈利的巨大落差**：PE超100倍，ROE仅3%，市场定价隐含的增长预期极高
2. **战略价值与财务回报的矛盾**：国家战略资产 vs 股东实际回报为零
3. **扩产必要性与资本效率的矛盾**：每年80亿美元资本开支 vs 50亿元净利润
4. **技术突破叙事与商业化现实**：7nm良率突破 vs DUV多重曝光的高成本
