# 海尔智家（6690.HK / 600690.SH）投资研究框架

> 研究日期：2026年5月16日 | 货币单位：人民币（除特别注明）

## 研究对象概览

海尔智家是全球化程度最深的中国家电企业，2025年营收首次突破3000亿元（3023.47亿元），海外收入1545.45亿元，占比51.1%，首次超过国内市场。公司旗下拥有海尔、卡萨帝、Leader、GE Appliances、Fisher & Paykel、Candy、AQUA七大全球品牌，在全球163个制造中心和126个营销中心运营。港股当前PE约9.1倍，ROE 16.97%，股息率约5-6.5%（取决于H/A股及计算口径）。

## 核心结论

**综合评级：值得深入跟踪，可逐步建仓 ★★★★☆（4/5）**

海尔智家是少数真正实现全球化自主品牌运营的中国制造企业。其优势在于：全球七品牌矩阵覆盖多元市场、本土化制造体系有效对冲地缘风险、冰洗品类国内份额接近50%构成强壁垒、现金流扎实（经营现金流为净利润1.33倍）、股东回报持续提升（分红比例从48%提至55%，规划2028年不低于60%）。

主要顾虑包括：北美关税政策不确定性对GE Appliances利润的冲击、国内白电市场弱复苏背景下增长动能放缓、"人单合一"模式在组织效率方面的争议、以及港股流动性折价问题。

9倍PE买入一个全球化家电龙头，安全边际相对充足，但需密切关注北美关税演进和国内需求恢复节奏。
