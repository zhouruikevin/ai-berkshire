## 公司简介

中国海洋石油有限公司是中国最大的海上油气生产商，也是全球最大的独立油气勘探及生产公司之一。公司主要从事海上原油及天然气的勘探、开发、生产及销售，业务遍及中国海域及海外多个国家和地区。母公司为中国海洋石油集团有限公司（国务院国资委直属央企）。

## 研究框架

本研究采用四维度并行分析：

1. **商业模式分析** — 聚焦生意本质、护城河、商业逻辑
2. **财务估值分析** — 聚焦盈利质量、资产负债表、估值安全边际
3. **行业竞争分析** — 聚焦产业格局、竞争优势、行业趋势
4. **风险与管理层评估** — 聚焦核心风险、管理层能力、公司治理
