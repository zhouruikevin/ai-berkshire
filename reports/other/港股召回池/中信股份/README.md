# 中信股份（0267.HK）投资研究

## 基本信息
- **公司名称**：中国中信股份有限公司
- **股票代码**：0267.HK
- **行业**：综合集团（金融+实业）
- **PE（TTM）**：5-6倍
- **ROE**：8-9%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 收入 | 7,693亿元 | +3.0% |
| 归母净利润 | 587亿元 | +0.9% |
| 末期股息 | 每股0.385元 | - |

## 投资亮点
1. PE仅5-6倍——与长实并列最低估值
2. ROE 8-9%——在低PE公司中相对较好
3. 中信银行+中信证券+中信建投=强综合金融
4. 政策红利（央企改革+市值管理）

## 核心风险
1. 综合集团折让严重
2. 银行业净息差持续收窄
3. 业务过于分散（金融+地产+制造+资源等）
4. 利润增速极慢（+0.9%）
