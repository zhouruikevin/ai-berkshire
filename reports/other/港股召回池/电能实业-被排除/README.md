# 电能实业（0006.HK）投资研究

## 基本信息
- **公司名称**：电能实业有限公司
- **股票代码**：0006.HK
- **行业**：电力/公用事业
- **PE（TTM）**：~22倍
- **ROE**：5-6%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 收入 | 7.71亿港元 | -16.1% |
| 股东应占溢利 | 62.36亿港元 | +2% |
| 全年每股派息 | 2.82港元 | 维持稳定 |
| 末期股息 | 每股2.04港元 | - |
| 股息率 | ~4.6% | - |

## 投资亮点
1. 与长江基建高度关联，资产质量稳定
2. 派息率极高（>90%），收息确定性好
3. 全球公用事业资产组合
4. 高评级信用，融资成本低

## 核心风险
1. PE 22倍对ROE 5-6%极贵
2. 收入大幅下滑16%
3. 增长接近停滞
4. 与长江基建高度重叠，缺乏独立投资价值
