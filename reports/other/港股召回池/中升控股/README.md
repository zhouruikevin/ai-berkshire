## 核心业务

中升控股是中国最大的豪华汽车经销商集团之一，主要经营：

1. **新车销售**：梅赛德斯-奔驰、宝马、奥迪、雷克萨斯等豪华品牌，以及丰田、本田、日产等中高端品牌；2024年底新增问界（AITO/鸿蒙智行）新能源品牌
2. **售后服务**：维修保养、零配件销售、延保服务
3. **二手车业务**：二手车交易及置换
4. **汽车金融及保险**：汽车贷款佣金、保险代理

## 研究背景

- **行业剧变**：新能源汽车渗透率突破50%，传统豪华品牌经销商遭遇价格倒挂、利润坍塌
- **业绩转折**：2025年首次录得净亏损16.73亿元，其中商誉减值22.91亿元
- **估值洼地**：2026E PE约4.3倍，市场对传统经销商模式的估值严重折价
- **转型关键期**：加速布局华为问界、吉利系等新能源品牌，目标2026年末新能源门店数量倍增
