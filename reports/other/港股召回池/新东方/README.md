## 公司简介

新东方教育科技集团是中国最大的民办教育服务提供商，由俞敏洪于1993年创立。2021年"双减"政策后，公司经历重大业务转型，剥离K-12学科类培训业务，转向素质教育、留学考培、成人教育及直播电商（东方甄选）等领域。

## 核心业务结构（2025-2026财年）

1. **教育业务**（核心）：留学考培、素质教育、智能学习系统
2. **东方甄选**（已剥离）：直播电商平台，2026财年上半年GMV 41亿元
3. **新教育业务**：同比增长35%，包括游学、STEAM等

## 关键财务数据

- 2025财年全年营收：49亿美元，同比+13.6%

- 2026财年营收指引：51.45-53.9亿美元（+5%-10%）
