# 长和（0001.HK）投资研究

## 基本信息
- **公司名称**：长和实业集团有限公司
- **股票代码**：0001.HK
- **行业**：综合集团
- **PE（TTM）**：~19倍
- **ROE**：2-3%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 呈报净利润 | 118.4亿港元 | （含3UK合并亏损105亿） |
| 基本净利润（剔除一次性） | 223亿港元 | +7% |
| 港口收益 | 489亿港元 | +8% |
| 零售收益 | 2,093亿港元 | +10% |

## 投资亮点
1. 全球多元化集团——港口、零售、基建、能源、电讯
2. 资产出售释放价值（3UK合并、UKPN出售等）
3. 净负债率极低（降至7%）
4. 李嘉诚家族长期资本配置能力

## 核心风险
1. ROE仅2-3%——13家中最低
2. 综合折让严重（市场厌恶复杂集团）
3. 3UK合并产生105亿港元一次性亏损
4. 业务过于分散，难以形成合力
