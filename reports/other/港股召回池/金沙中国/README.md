# 金沙中国（1928.HK）投资研究

## 基本信息
- **公司名称**：金沙中国有限公司
- **股票代码**：1928.HK
- **行业**：博彩/度假村
- **PE（TTM）**：15-18倍
- **ROE**：20-25%

## 核心财务数据
| 指标 | 数值 | 说明 |
|------|------|------|
| 2025年净收入 | ~78亿美元 | 四季度同比+16% |
| 净利润 | ~10亿美元 | 同比-3.3% |
| EBITDA利润率 | ~29-33% | 运气调整后~29% |
| 物业 | 5大综合度假村 | 10,829间客房 |
| 赌台/角子机 | ~1,680张/3,700台 | - |
| 2025股息 | 每股3.7美仙 | 股息率~1.6% |

## 投资亮点
1. ROE 20-25%——13家公司中最高
2. 澳门博彩牌照垄断（6家之一）
3. 旅游/消费复苏利好
4. 母公司LVS全球运营经验

## 核心风险
1. 博彩监管政策突变风险
2. 中国经济放缓压制贵宾厅收入
3. 新加坡/东南亚竞争分流
4. 资本密集——物业维护/翻新需持续投入
