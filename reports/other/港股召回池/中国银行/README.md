# 中国银行（3988.HK）投资研究

## 基本信息
- **公司名称**：中国银行股份有限公司
- **股票代码**：3988.HK
- **行业**：国有大行
- **PE（TTM）**：~6倍
- **ROE**：~9%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 6,583亿元 | +4.48% |
| 归母净利润 | 2,430亿元 | +2.18% |
| 净息差 | 1.26% | -14BP |
| 非利息收入占比 | 33.06% | 提升 |
| 全年股息 | 每10股2.263元 | - |

## 投资亮点
1. PE仅6倍——极度低估的大型银行
2. ROE 9%——在低PE公司中属于优秀
3. 国际化程度最高的中资银行
4. 稳定的股息回报

## 核心风险
1. 净息差持续收窄（-14BP），核心盈利能力下降
2. 地产相关不良贷款风险
3. 国有大行受政策任务影响（让利实体经济）
4. 利润增速偏慢（+2.18%）
