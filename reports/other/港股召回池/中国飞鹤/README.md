## 研究背景

中国飞鹤是国内婴幼儿配方奶粉市场龙头，连续多年市占率第一。公司面临出生人口持续下降的结构性逆风，2025年营收181.1亿元（同比-12.7%），净利润19.4亿元（同比-45.7%），为近六年最差业绩。当前PE约13-15倍，股息率超6%，正处于"高分红+强回购"阶段。核心问题：第二曲线能否承接主业下滑？低估值是价值陷阱还是机会？
