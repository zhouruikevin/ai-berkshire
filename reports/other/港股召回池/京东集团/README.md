# 京东集团（9618.HK）投资研究

## 基本信息

| 项目 | 内容 |
|------|------|
| 公司名称 | 京东集团 |
| 股票代码 | 9618.HK / JD (NASDAQ) |
| 行业 | 自营电商 + 物流 + 即时零售 |
| 2025年营收 | 13,091亿元（同比+13%） |
| 2026Q1营收 | 3,157亿元（同比+4.9%） |
| 2025年Non-GAAP净利润 | 270亿元 |
| 当前PE（TTM） | ~10倍 |

| 员工总数 | 90万+ |

## 研究框架

本研究采用四维度并行分析框架：

1. **商业模式分析** — 生意本质、护城河、用户价值
2. **财务估值分析** — 盈利质量、资本配置、安全边际
3. **行业竞争分析** — 竞争格局、产业趋势、生态位
4. **风险管理层评估** — 核心风险、管理层能力、治理结构

## 核心投资变量

- 外卖业务：2025年投入近300亿，目标2026年市占率30%，是利润最大变量
- 物流壁垒：京东物流2025年营收2171亿（+18.8%），一体化供应链客户超9万家
- 股东回报：2025年回购30亿美元+分红14亿美元，总回报率~10%
- 竞争格局：电商份额被抖音侵蚀，从第二滑至第三（GMV口径）
- 增长放缓：2026Q1营收增速降至4.9%，家电国补退坡影响显现
