## 投资背景

理想汽车是中国领先的新能源汽车企业，以增程式电动SUV起家，凭借精准的产品定义和家庭用户定位，在2023-2024年实现高速增长。但2025年遭遇重大挫折：MEGA纯电旗舰失利、增程市场竞争加剧、全年交付同比下滑近19%，净利润暴跌86%。2026年公司正处于"增程+纯电"双轮驱动转型期，面临激烈竞争和盈利能力修复的双重挑战。
