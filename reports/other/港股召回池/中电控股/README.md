## 研究背景

中电控股是香港两大电力供应商之一，为九龙、新界及大部分离岛约283万客户供电，供电可靠性达99.999%。香港电力业务受《管制计划协议》保障，准许回报率为固定资产净值的8%，协议有效期至2033年。

公司业务遍及香港、中国内地、澳洲、印度及东南亚，是亚太区领先的电力企业。控股股东为嘉道理家族，持股约35%，家族经营超过百年。
