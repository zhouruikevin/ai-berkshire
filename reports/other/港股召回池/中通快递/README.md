## 研究背景

中通快递是中国快递行业业务量第一、盈利能力最强的企业，连续十年保持市场份额领先。公司采用"终端加盟+分拣直营"的模式，以成本优势构建竞争壁垒。当前PE约13倍，ROE约17%，同时推出35亿美元累计回购计划。核心隐忧在于快递行业价格战的持续性以及市场份额的波动。
