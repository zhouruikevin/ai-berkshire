## 投资论点预览

**看多逻辑：** 汽车业务从0到千亿仅用2年，"人车家"生态协同开始兑现，2026年YU7放量有望推动利润跳升。

**看空逻辑：** 手机毛利率承压（8.3%），汽车竞争白热化，严重依赖雷军个人，印度市场萎缩，估值已部分反映乐观预期。
