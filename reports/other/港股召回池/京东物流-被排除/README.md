# 京东物流（2618.HK）投资研究

## 基本信息
- **公司名称**：京东物流股份有限公司
- **股票代码**：2618.HK
- **行业**：供应链物流
- **PE（TTM）**：~15倍
- **ROE**：~12%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 2,171亿元 | +18.8% |
| 经调整净利润 | 77.1亿元 | 大幅增长 |
| 上半年一体化供应链收入 | 501亿元 | +19.9% |
| 外部客户收入占比 | 67.1% | 持续提升 |
| 一线员工数 | 超66万人 | - |

## 投资亮点
1. 收入增速强劲（18.8%），突破2000亿元
2. 一体化供应链模式形成竞争壁垒
3. 外部客户收入占比持续提升，去京东化
4. AI/自动化降本增效空间大

## 核心风险
1. 人力密集型企业，66万+员工，成本刚性
2. 毛利率偏低（~9-10%），利润薄
3. 与京东集团关联交易依赖
4. 快递行业价格战风险
