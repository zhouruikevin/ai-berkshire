## 研究背景

新奥能源是港股燃气板块的标杆民企，PE 9.9倍、ROE 15%的组合在公用事业中属于优质水平。当前最大变量是新奥股份（600803.SH）对其发起的私有化收购：每股80港元对价（24.5港元现金+2.94股新奥股份H股），交易总规模约599亿港元。私有化完成与否、完成后的整合效果，将深刻改变公司的投资逻辑。

本研究以"正反两面、客观不预设"为原则，逐一拆解商业模式、财务估值、行业竞争、风险管理层四个维度。
