## 核心结论

**综合评分：★★★★☆（4/5）**

创科实业是港股最稀缺的全球制造业龙头之一。Milwaukee品牌定位专业级市场，形成了"电池平台锁定+持续新品扩展"的飞轮效应；Ryobi覆盖消费级市场，通过Home Depot独家渠道稳定增长。公司2025年营收153亿美元，自由现金流14亿美元，净现金7亿美元，资产负债表极为健康。毛利率从2015年的35.7%稳步提升至2025年的41.2%，体现品牌溢价能力持续增强。

主要风险在于：（1）美国关税政策的不确定性，虽已基本完成中国对美出口的产能转移，但局势仍在变化；（2）美国住房市场周期性波动对需求的影响；（3）新CEO Steven Richman上任不足两年，需观察执行力。

当前股价约124.5港元（2026年5月），对应2025年PE约23倍。考虑到公司的龙头地位、持续的利润率改善和强劲的自由现金流，估值合理但谈不上便宜。建议**观望等待更好价格，或在100港元以下积极买入**。

**投资建议：观望（等待安全边际）**
