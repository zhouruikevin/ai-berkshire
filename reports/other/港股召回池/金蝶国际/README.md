## 研究背景

金蝶国际是中国企业级SaaS领域的龙头公司之一，经历了从财务软件到ERP再到云SaaS的三次战略转型。公司于2025年首次实现扭亏为盈，标志着云转型进入收获期。当前市值约293亿港元，对应2025年归母净利润PE约150倍+。公司提出"AI优先、订阅优先、全球化"三大核心战略，2026年AI套件收入目标10亿元。
