## 投资背景

同程旅行是中国第三大OTA平台（按GMV计约15%市场份额），独占微信钱包火车票/机票和酒店两大入口，在非一线城市拥有超87%的用户占比。公司正从"下沉市场OTA"向"全场景旅行平台"转型，同时推进国际化和AI智能化布局。

## 研究框架

本研究采用四大师分析框架，从不同投资视角进行深度研究：

1. **[商业模式分析 - 段永平视角](01-商业模式分析-段永平视角.md)**：聚焦商业本质、用户价值、长期竞争力
2. **[财务估值分析 - 巴菲特视角](02-财务估值分析-巴菲特视角.md)**：聚焦财务质量、盈利能力、估值安全边际
3. **[行业竞争分析 - 芒格视角](03-行业竞争分析-芒格视角.md)**：聚焦竞争格局、行业演变、逆向思维
4. **[风险管理层评估 - 李录视角](04-风险管理层评估-李录视角.md)**：聚焦管理层品质、风险敞口、长期结构性因素
