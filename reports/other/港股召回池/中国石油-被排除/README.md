# 中国石油（0857.HK）投资研究

## 基本信息
- **公司名称**：中国石油天然气股份有限公司
- **股票代码**：0857.HK
- **行业**：油气勘探/炼化
- **PE（TTM）**：10-11倍
- **ROE**：~5%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 28,645亿元 | -2.5% |
| 归母净利润 | 1,573亿元 | -4.5% |
| 末期股息 | 每股0.25元 | 派息率54.7% |

## 投资亮点
1. 国内最大油气生产商，上游资源禀赋强
2. 天然气业务高速增长，符合能源转型方向
3. 估值相对便宜（PE 10-11倍）
4. 稳定分红，股息率约5%

## 核心风险
1. 油价下行影响上游利润
2. 炼化板块盈利能力弱
3. ROE偏低（~5%），资本效率不高
4. 海外资产面临地缘政治风险
