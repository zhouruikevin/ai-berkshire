## 公司简介

百威亚太是全球最大啤酒集团百威英博（AB InBev）的亚太区业务平台，2019年在港交所上市。公司运营百威、科罗娜、哈尔滨、福佳等多个啤酒品牌，业务覆盖中国、韩国、印度、越南等亚太市场。

## 核心业务结构（2025年）

1. **亚太西部（含中国）**：收入占比约65%，中国为最大单一市场
2. **亚太东部（韩国、日本）**：收入占比约15%
3. **南亚（印度、越南）**：收入占比约20%，增长最快

## 关键财务数据（2025年）

- 全年收入：57.6亿美元，同比-6.1%
- 正常化EBITDA：15.9亿美元，同比-9.8%
- 归母净利润：4.89亿美元，同比-32.6%
- 分红：7.5亿美元（派息率153%）
- 中国区销量：同比-3.9%，吨价同比-7.7%
