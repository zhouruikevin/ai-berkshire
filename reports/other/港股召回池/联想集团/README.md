## 公司简介

联想集团成立于1984年，由中国科学院计算技术研究所投资创办。2004年收购IBM全球PC业务，成为全球化企业。目前是全球最大的PC厂商（市占率25.4%），同时在服务器/基础设施（ISG）和解决方案/服务（SSG）领域持续扩张。

三大业务板块：
- **IDG（智能设备业务集团）**：PC、平板、手机等，核心利润来源
- **ISG（基础设施方案业务集团）**：服务器、存储、AI基础设施，高速增长但尚未稳定盈利
- **SSG（方案服务业务集团）**：IT服务与解决方案，高利润率（22%+运营利润率）

## 研究背景

全球PC龙头，PE 10-12倍，ROE约20%（高杠杆驱动），低利润率的组装生意。近年AI浪潮下，ISG服务器业务高速增长，AI营收占比升至32%。公司提出两年内营收破1000亿美元、净利润率超5%的目标。值得深入研究其商业模式的可持续性、AI转型的真实含金量、以及当前估值是否充分反映了基本面。
