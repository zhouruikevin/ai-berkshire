# 中国石化（0386.HK）投资研究

## 基本信息
- **公司名称**：中国石油化工股份有限公司
- **股票代码**：0386.HK
- **行业**：石油炼化
- **市值**：约4,500亿港元
- **PE（TTM）**：15-17倍
- **ROE**：6-7%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 2,784亿元 | -9.4% |
| 归母净利润 | 318亿元 | -36.8% |
| 基本每股收益 | 0.26元 | - |
| 全年股息 | 每股0.2元 | 派息率81% |
| 经营现金流净额 | 162.5亿元 | +8.8% |

## 投资亮点
1. 国内最大炼化企业，产业链一体化优势
2. 高派息率（81%），股息收益率可观
3. 加油站网络形成天然护城河
4. 向新能源转型积极布局（充电桩、氢能）

## 核心风险
1. 国际油价波动直接影响盈利
2. 成品油需求见顶（新能源车替代）
3. 化工产品价格周期性强
4. 国企治理效率待提升
