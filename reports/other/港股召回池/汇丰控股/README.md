## 研究框架

本研究从四个维度对汇丰控股进行深度分析：

1. **商业模式分析**（段永平视角）：聚焦护城河、商业本质、长期可持续性
2. **财务估值分析**（巴菲特视角）：聚焦盈利质量、资本回报、安全边际
3. **行业竞争分析**（芒格视角）：聚焦行业格局、竞争优势、多元思维模型
4. **风险管理层评估**（李录视角）：聚焦管理层品质、风险敞口、制度环境

## 核心数据速览（2025年报）

- 营业收入：683亿美元（+4% YoY）
- 税前利润：299亿美元（-7.4% YoY）
- 税后利润：231亿美元（-19亿美元 YoY）
- 全年股息：每股0.75美元
- 股份回购：60亿美元（2025年完成）
- 2026目标：银行净利息收入≥450亿美元，ROTE≥17%
