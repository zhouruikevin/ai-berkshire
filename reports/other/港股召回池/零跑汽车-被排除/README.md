## 核心投资论点

**多头论点：**
- 全域自研带来的成本优势（自研零部件占整车成本65%）
- 2025年首次全年盈利，验证了商业模式
- Stellantis战略合作打开海外市场
- 新势力销冠地位，规模效应持续释放
- 2026年目标100万辆/50亿净利润

**空头论点：**
- 2026年Q1毛利率跌至9.4%，净亏损3.9亿
- 单车利润极薄（2025年单车仅赚905元）
- 100万辆目标激进，执行压力大
- 价格战持续，比亚迪/吉利等竞争加剧
- 智驾能力相对落后第一梯队
