# 零跑汽车（9863.HK）商业模式分析

> 分析维度：生意本质、用户价值、差异化壁垒、长期可持续性
> 核心问题：这是不是一门好生意？护城河在哪里？能否长期持续？

---

## 一、生意的本质是什么？

零跑汽车的生意本质是：**以全域自研的垂直整合能力，在10-20万元主流价格带，提供高性价比的智能电动汽车。**

拆解来看：
- **收入模式**：卖车（占收入95%以上）+ 少量配件和服务
- **价值主张**：同价位配置最高，高价位功能下放到大众价位
- **核心能力**：自研电驱、电池PACK、智能座舱、智驾系统、热管理等核心零部件
- **成本结构**：自研零部件占整车成本约65%，省去供应商利润空间

与传统车企和其他新势力相比，零跑的商业模式更接近比亚迪的垂直整合路线，但规模远小于比亚迪。与理想、蔚来等走品牌溢价路线的新势力不同，零跑走的是"成本领先+技术平权"的路线。

## 二、用户价值分析

### 零跑为用户创造了什么价值？

1. **性价比极致**：C11、C16等车型在15-20万价位提供了远超同价位的配置水平（激光雷达、大电池、高算力芯片）
2. **技术平权**：将20-30万价位才有的智驾、智舱功能下放到10-15万级别
3. **产品一致性好**：因为核心零部件自研，供应链稳定性和品控相对可控

### 用户画像

零跑的核心用户是价格敏感但对科技有追求的务实消费者，集中在二三线城市，年龄30-45岁，家庭用车需求为主。这个用户群体规模庞大，但品牌忠诚度相对较低——他们买的是"值"，而非"牌子"。

### 价格带的战略意义

10-20万元是中国乘用车市场最大的价格带，占总销量的40%以上。零跑选择这个战场，天花板很高，但竞争也最为惨烈。比亚迪、吉利、长安、奇瑞都在这里厮杀。

## 三、全域自研——护城河还是负担？

### 正面：自研带来的结构性优势

1. **成本优势**：自研电驱系统比外采节约约10%成本；自研电池PACK省去宁德时代的中间利润
2. **迭代速度**：电子电气架构自研，OTA升级不受供应商制约
3. **差异化**：CTC电池底盘一体化技术、四叶草中央域控架构，都是自研独有
4. **毛利保护**：当行业打价格战时，自研企业有更多降价空间

零跑2025年毛利率达14.5%，在10万级均价的车企中已属优秀。作为参照，小鹏（均价20万+）毛利率也才14-15%。

### 反面：自研的风险和挑战

1. **研发投入持续性**：每年需要50亿以上研发投入，一旦销量不及预期，研发摊销将侵蚀利润
2. **技术广而不精**：什么都自研，意味着每个领域的投入都有限。智驾方面确实落后于华为、小鹏
3. **规模门槛**：自研模式在规模较小时（<30万辆/年）效率不如外采，只有规模做大才能摊薄成本
4. **人才瓶颈**：5000名工程师要覆盖电池、电驱、智驾、座舱、底盘等所有领域，分散程度高

### 与比亚迪垂直整合的关键区别

比亚迪的垂直整合建立在年销400万辆以上的规模基础上，且电池（弗迪）、半导体等都可以外供赚取额外利润。零跑的垂直整合目前只服务于自身，规模效应尚未完全释放。如果零跑能做到100万辆以上年销量，这个模式才会真正体现出优势。

## 四、Stellantis合作——第二增长曲线？

### 合作模式

Stellantis持有零跑20%股权，双方成立零跑国际合资公司，由Stellantis负责零跑产品在海外的销售和服务网络。这个模式的精妙之处在于：

- 零跑不需要自建海外渠道（省去巨大的固定成本投入）
- 借助Stellantis在欧洲的6000+经销商网络
- 西班牙工厂本地化生产，规避关税风险
- Stellantis的品牌背书降低了海外消费者的信任门槛

### 进展评估

- 2025年出口6.7万辆，位居新势力第一
- 2026年Q1海外交付4.09万辆，占总量37%，增长迅猛
- 西班牙工厂预计2026年投产，首款车型B10

### 风险

- Stellantis自身经营困难（2024年利润大幅下滑），合作持续性存疑
- 欧洲电动车补贴政策不确定
- 地缘政治风险（关税壁垒、反补贴调查）

## 五、商业模式的可持续性评估

### 正向循环（飞轮效应）

规模越大 → 自研成本摊薄 → 价格更有竞争力 → 销量进一步增长 → 更多研发投入 → 产品力提升

这个飞轮在2024-2025年已经初步转起来：销量从29万到60万翻倍，毛利率从约10%提升到14.5%，首次实现盈利。

### 负向风险（飞轮停转的可能）

- 如果100万辆目标达不到（比如只卖70-80万），研发摊销压力会显著增大
- 价格战持续导致毛利率回落（2026年Q1毛利率已跌至9.4%）
- 新车型（A10、D19等）如果表现不及预期，存量车型又面临老化

### 长期终局的思考

中国新能源汽车市场最终可能只剩5-8家主要玩家。零跑能否成为其中之一？

**有利因素**：已过盈亏平衡点、自研体系完整、海外渠道已建立、现金储备充裕（378.8亿）

**不利因素**：品牌力弱于比亚迪/理想/华为、单车利润极薄（905元/辆）、在智驾军备竞赛中处于追赶位置

## 六、总结评分

| 评估维度 | 评分 | 说明 |
|----------|------|------|
| 生意模式清晰度 | ★★★★☆ | 全域自研+性价比路线逻辑清晰 |
| 用户价值创造 | ★★★★☆ | 确实为用户提供了超值产品 |
| 护城河深度 | ★★★☆☆ | 自研有壁垒但规模还不够形成深沟 |
| 可持续性 | ★★★☆☆ | 依赖持续放量，飞轮是否能转快仍需验证 |
| 商业模式综合评估 | ★★★☆☆ | 方向正确但未到安全区，仍处关键爬坡期 |

### 核心结论

零跑的商业模式逻辑自洽——全域自研→成本领先→性价比产品→规模放量→正向循环。但这个模式对"规模"的依赖度极高。2025年60万辆已证明模式可行，2026年能否达到100万辆将是决定性验证。如果能持续放量至100万+，这将是一个优秀的生意；如果卡在60-80万辆区间，则利润空间将被研发投入和价格战持续挤压。

**一句话总结**：好的方向，但还没到可以说"好生意"的阶段。护城河正在修建中，尚未完工。
