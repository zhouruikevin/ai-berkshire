# 长实集团（1113.HK）投资研究

## 基本信息
- **公司名称**：长实集团有限公司
- **股票代码**：1113.HK
- **行业**：地产/投资
- **PE（TTM）**：5-6倍
- **ROE**：3-4%

## 核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 2025上半年净利润 | 63亿港元 | -26.75% |
| 物业销售确认收入（上半年） | 73.7亿港元 | +58.9% |
| 负债净额/总资本 | ~5.0% | 行业极低 |
| 信用评级 | A/稳定（标普）、A2（穆迪）| - |

## 投资亮点
1. PE仅5-6倍——13家中最低
2. 负债率极低（5%），财务极为安全
3. 大量低成本土地储备（早期获取）
4. 多元化布局（地产+基建+酒店+飞机租赁）

## 核心风险
1. ROE仅3-4%——极低资本回报率
2. 香港/内地房地产市场持续低迷
3. 利润大幅下滑26.75%
4. 物业重估价值下降
