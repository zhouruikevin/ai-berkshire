# 中国电信（0728.HK）投资研究

## 基本信息
- **公司名称**：中国电信股份有限公司
- **股票代码**：0728.HK
- **行业**：固网/移动通信
- **PE（TTM）**：~16倍
- **ROE**：~7%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 5,239亿元 | +0.7% |
| 服务收入 | 4,854亿元 | +0.7% |
| 归母净利润 | 332亿元 | +0.5% |
| 天翼云收入 | 1,207亿元 | 快速增长 |
| 全年股息 | 每股0.272元 | 派息率75% |

## 投资亮点
1. 天翼云/AI数据中心增长强劲（AIDC收入345亿元）
2. 75%高派息率，股息收益率可观
3. 从传统运营商向"科技型企业"转型
4. 低估值+稳定现金流

## 核心风险
1. 传统通信业务增速见顶
2. 云业务面临阿里云、华为云激烈竞争
3. 5G大规模投资但ARPU提升有限
4. ROE仅7%，资本回报率偏低
