## 核心定位

亚太区最大的房地产投资信托基金，以香港社区零售商场为核心，辅以中国内地、澳大利亚、新加坡及英国的零售、办公及物流资产。物业组合地理分布：香港74.4%、中国内地13.8%、海外11.8%。
