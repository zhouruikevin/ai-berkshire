# 长江基建（1038.HK）投资研究

## 基本信息
- **公司名称**：长江基建集团有限公司
- **股票代码**：1038.HK
- **行业**：基础设施
- **PE（TTM）**：~20倍
- **ROE**：~6%

## 核心财务数据
| 指标 | 数值 | 说明 |
|------|------|------|
| 收入 | ~48亿港元 | 同比+7.2% |
| 归母净利润 | ~86亿港元（预期） | 稳健增长 |
| 中期股息 | 每股0.73港元 | +1.4% |
| 派息连续增长 | 28年 | 自上市以来从未减派 |

## 利润地理分布
| 地区 | 溢利贡献 | 占比 |
|------|----------|------|
| 英国 | 39.8亿港元 | 42.3% |
| 电能实业 | 22.0亿港元 | 23.4% |
| 澳洲 | 17.8亿港元 | 18.9% |
| 欧洲大陆 | 6.0亿港元 | 6.4% |
| 加拿大 | 5.3亿港元 | 5.6% |

## 投资亮点
1. 28年连续股息增长——港股最长连增纪录之一
2. 全球化基建资产组合，分散地理风险
3. 受监管资产提供可预测的稳定收益
4. 李嘉诚家族长期管理经验

## 核心风险
1. PE 20倍对ROE 6%偏贵
2. 英国资产占比过高（42%），脱欧/政策风险
3. 利率上升增加融资成本
4. 海外监管政策变动风险
