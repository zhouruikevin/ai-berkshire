## 研究框架

1. **商业模式分析**（段永平视角）：互联网保险的本质、差异化
2. **财务估值分析**（巴菲特视角）：盈利质量、成长性、估值
3. **行业竞争分析**（芒格视角）：保险科技格局、护城河
4. **风险管理层评估**（李录视角）：管理层、科技风险

- 总保费：357.35亿元（+6.9% YoY）
- 经调整归母净利润：18.00亿元（+198.3% YoY）
- 承保利润：连续第五年盈利

- 自营渠道保费：75.57亿元（占比21.1%）
