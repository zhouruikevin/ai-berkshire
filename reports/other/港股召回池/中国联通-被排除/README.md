# 中国联通（0762.HK）投资研究

## 基本信息
- **公司名称**：中国联合网络通信股份有限公司
- **股票代码**：0762.HK
- **行业**：移动通信
- **PE（TTM）**：~15倍
- **ROE**：~5.7%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 3,922亿元 | +0.7% |
| 服务收入 | 3,477亿元 | +0.5% |
| 归母净利润 | 91.3亿元（A股）/ 股东应占208亿元（港股） | +1.0-1.1% |
| 净资产收益率 | 5.7% | 稳定 |
| 5G专网收入 | 123亿元 | +51% |

## 投资亮点
1. 混改红利尚存，与BAT深度合作
2. 5G专网/物联网/车联网高速增长
3. 算力投资占比提升至35%+
4. 估值在三大运营商中相对合理

## 核心风险
1. ROE仅5.7%，三大运营商中最低
2. 移动用户规模最小（3.4亿），规模劣势明显
3. 营收增速几乎停滞（+0.7%）
4. 核心利润偏薄，增长动力不足
