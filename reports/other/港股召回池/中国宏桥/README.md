## 公司简介

中国宏桥是全球规模最大的电解铝生产商之一，由"山东首富"张士平创立，现由其子张波掌舵。公司构建了从铝土矿开采、氧化铝生产、自备电力到电解铝冶炼的完整一体化产业链，电解铝产能约646万吨，氧化铝产能约2100万吨。2025年实现营收1623.54亿元，净利润226.36亿元。

## 核心投资论点

- **全球最低成本电解铝生产商之一**：2024年吨铝成本约13232元，远低于行业平均
- **产能天花板政策赋予稀缺性**：中国电解铝产能上限4500万吨，运行产能已达4410万吨，新增空间极小
- **需求端结构性增长**：新能源汽车、光伏、储能、AI数据中心带来新增铝需求
- **高分红属性**：2025年分红比率约65%，末期股息每股1.65港元
- **海外资源储备深厚**：几内亚铝土矿+西芒杜铁矿+印尼氧化铝
