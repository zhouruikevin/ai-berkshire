# 中国中车（1766.HK）投资研究

## 基本信息
- **公司名称**：中国中车股份有限公司
- **股票代码**：1766.HK
- **行业**：轨道交通装备
- **PE（TTM）**：8-9倍
- **ROE**：~8%

## 2025年度核心财务数据
| 指标 | 数值 | 同比变化 |
|------|------|----------|
| 营业收入 | 2,731亿元 | +10.79% |
| 归母净利润 | 131.8亿元 | +6.40% |
| 新签订单 | 3,461亿元 | 强劲 |
| 海外收入 | 348亿元 | +22.88% |
| 海外收入占比 | 12.75% | 持续提升 |

## 业务结构
| 板块 | 收入 | 同比 |
|------|------|------|
| 铁路装备 | 1,236亿元 | +11.9% |
| 新产业 | 1,031亿元 | +19.4% |
| 城轨与基础设施 | 421亿元 | -7.4% |

## 投资亮点
1. 全球轨交装备龙头——市场份额全球第一
2. 新产业收入突破千亿——第二增长曲线成型
3. 海外收入增长23%——国际化加速
4. PE仅8-9倍——对增长性而言便宜

## 核心风险
1. 国内铁路投资高峰已过
2. 净利率偏低（~4.8%），利润薄
3. 海外业务面临地缘政治阻力
4. 城轨业务下滑7.4%
