## 核心观察

**正面因素：** 全球化扩张势头强劲，IP战略转型成效显著，供应链优势深厚，高ROE+高股东回报。

**负面因素：** 永辉投资拖累报告利润，资产负债率因收购大幅攀升，国内同店增长承压，加盟模式争议。
