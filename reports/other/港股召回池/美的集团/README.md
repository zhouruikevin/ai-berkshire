## 公司概况

美的集团是中国最大的家电制造企业，业务覆盖智能家居（空调、冰箱、洗衣机、厨电等）和商业及工业解决方案（楼宇科技、机器人与自动化、新能源及工业技术）两大板块。公司在全球200多个国家和地区拥有约400家子公司，海外营收占比超40%。
