## 投资背景

比亚迪是全球新能源汽车龙头，2025年纯电动车销量超越特斯拉成为全球第一。公司采用垂直整合模式，覆盖电池、电机、电控、半导体等核心零部件。当前处于全球化扩张期，2026年海外销量目标130万辆。主要风险包括价格战压制利润率、欧盟关税壁垒、地缘政治不确定性。
