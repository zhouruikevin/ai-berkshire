## 公司简介

康师傅控股有限公司由台湾魏应州家族于1992年在天津创立，是中国最大的方便面和饮料企业之一。核心品牌"康师傅"在中国家喻户晓。2018年创始人魏应州退休，长子魏宏名接任董事会主席；2026年1月三子魏宏丞接任CEO。

## 核心业务结构（2025年）

1. **饮品业务**：营收约501亿元（占比63%），包括茶饮料、果汁、包装水
2. **方便面业务**：营收约284亿元（占比36%），包括容器面、袋面
3. **其他**：约1%

## 关键财务数据（2025年）

- 全年营收：约791亿元，同比增长
- 归母净利润：约45亿元，同比+20.5%
- 整体毛利率：34.8%（+1.7pct）
- ROE：30.8%（+4.2pct）
- 全年派息：0.66元/股（派息率约100%）
