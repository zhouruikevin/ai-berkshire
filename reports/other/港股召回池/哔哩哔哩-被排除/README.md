## 研究背景

哔哩哔哩是中国Z世代视频社区平台，以弹幕文化和PUGV（专业用户生成视频）内容为核心。2025年首次实现全年盈利，但商业化效率相比抖音、快手仍有较大差距。当前PE约30倍（调整后净利润口径），市场分歧在于：B站的社区属性是商业化的天花板，还是尚未释放的价值金矿？
