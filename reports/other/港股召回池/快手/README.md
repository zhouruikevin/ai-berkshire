## 研究背景

快手是中国第二大短视频平台，仅次于抖音。公司在2025年实现了营收1428亿元、经调整净利润206亿元的历史最佳业绩，但股价在年报发布后反而暴跌14%，市场给出的信号非常明确：**利润的质量不如利润的增长重要**。

核心矛盾在于：
- **用户增长见顶**：MAU增速降至2%以下，Q4环比流失800万DAU
- **电商增速下行**：GMV增速从2021年的78%降至2025年的15%，公司宣布不再披露GMV
- **可灵AI是唯一亮点**：ARR从2025年底的2.4亿美元飙升至2026年4月的5亿美元，传闻分拆估值200亿美元
- **估值便宜但有原因**：PE 11-12倍，低于抖音集团和腾讯，但增长叙事面临挑战
