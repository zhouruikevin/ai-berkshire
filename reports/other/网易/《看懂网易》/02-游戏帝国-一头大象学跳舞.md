# 游戏帝国——一头大象学跳舞

> 《看懂网易》系列 · 第 02 篇 · 核心业务
> 阅读时间约 10 分钟

---

## 一组让你重新认识网易游戏的数字

先看网易游戏在中国游戏行业里的位置：

| 指标 | 数据 | 说明 |
|------|------|------|
| 2025年游戏收入 | 921亿元（+11%） | 中国第二，仅次于腾讯 |
| 占网易总营收 | 82% | 绝对核心，公司命脉 |
| 运营最久的产品 | 梦幻西游（2003年上线，23年） | 同时在线创纪录358万 |
| 最大的出海尝试 | 漫威争锋 | 首月全球收入$1.36亿 |
| 海外收入占比 | ~10% | 腾讯为30% |
| 研发投入 | 177亿元（占营收15.7%） | 重研发基因 |

921亿的游戏收入，11%的增速——放在全球游戏行业里都是顶级水准。但这头大象有一个明显的特征：**它在国内跳得很好，但在国际舞台上还在学步。**

这一篇要回答的核心问题是：**网易游戏的921亿能不能继续增长？增长从哪里来？**

---

## 一、长青产品：网易游戏的"永动机"

网易游戏最让人惊叹的不是爆款能力——而是**让一款游戏活20年以上还能创新高的能力**。

### 梦幻西游：23年不老的现金牛

| 指标 | 数据 |
|------|------|
| 上线时间 | 2003年 |
| 运营年限 | 23年 |
| 同时在线峰值 | **358万（历史纪录）** |
| 商业模式 | 点卡+道具混合付费 |
| 覆盖平台 | 端游 + 手游 |

一款2003年上线的游戏，在2025年创下了同时在线的历史纪录——这在全球游戏行业里几乎找不到第二个案例。

梦幻西游为什么能活这么久？三个原因：

**第一，社交绑定极深。** 梦幻西游的核心玩法围绕帮派、结婚、师徒等社交系统展开。一个玩了十几年的玩家，帮派里有朋友、游戏里有"老婆"、账号上有数万元的虚拟资产——迁移成本极高。

**第二，经济系统形成了自循环。** 梦幻西游有一套极其复杂的虚拟经济体系——玩家之间的道具交易、宝石合成、装备锻造形成了自运转的经济循环。这套系统越成熟，玩家越难离开。

**第三，持续的内容更新。** 23年里从未间断的版本更新、新资料片、新玩法——让老玩家保持新鲜感，也不断吸引新玩家。

**反方观点**：再强的长青游戏也有生命周期。梦幻西游的核心用户群在老化——30-40岁的主力玩家群体消费能力虽强，但时间越来越少。年轻用户对这类"古典"网游的兴趣在下降。同时在线358万可能已经接近天花板而非起点。

### 永劫无间：全球化潜力的试金石

| 指标 | 数据 |
|------|------|
| 上线时间 | 2021年 |
| 四周年成绩 | Steam全球热销榜登顶 |
| 游戏类型 | 动作对战（格斗+大逃杀） |
| 特点 | 亚洲风格的动作美学 |

永劫无间证明了网易在全球市场上不是没有竞争力——它的动作系统在全球玩家中获得了认可。四周年登顶Steam全球热销，说明这款产品的生命力仍在。

但永劫无间也暴露了网易的一个结构性短板：**单点成功难以复制成系统性的全球化能力。** 永劫无间在海外的成功更多来自产品本身的品质，而非网易的全球化发行网络和运营体系。

### 蛋仔派对：年轻化的关键一步

蛋仔派对是网易在"全年龄向"赛道上的重要布局——面向年轻用户和女性用户，风格轻松可爱，用户创作内容丰富。这款产品填补了网易在轻度休闲领域的空白。

但蛋仔派对面临的竞争也很激烈——它本质上在和腾讯的小程序游戏、以及各种轻度休闲游戏抢同一批用户的时间。

## 二、漫威争锋：最大的变量

如果说整个网易游戏的故事里只有一个变量值得跟踪，那就是漫威争锋。

### 数据全景

| 阶段 | 数据 | 说明 |
|------|------|------|
| 上线首月 | 全球收入$1.36亿 | 2025年最亮眼的新游戏之一 |
| 累计玩家 | 4,000万 | 短时间内积累的用户量惊人 |
| 同时在线峰值 | 64万 | 上线初期 |
| 同时在线低谷 | **9.6万** | 峰值暴跌85% |
| 同时在线回升 | **20万+** | 版本更新后回升 |
| Steam好评率 | 63% | "褒贬不一"——不算好 |

### 这组数据怎么读？

**首月$1.36亿——说明漫威品牌的号召力巨大。** 漫威是全球最有价值的娱乐品牌之一，网易拿到这个品牌的游戏开发权本身就是一种能力证明。

**同时在线从64万暴跌到9.6万——说明首发体验有问题。** 85%的同时在线流失率在竞技游戏里属于严重警报。大量玩家尝鲜后离开，通常意味着游戏的核心玩法循环（游戏节奏、匹配体验、角色平衡等）没有达到预期。

**从9.6万回升到20万+——说明团队在快速修正。** 版本更新和运营调整把一部分玩家拉了回来。这是积极信号，证明问题不是"不可修复的"。

**63%的Steam好评率——说明仍有大量不满。** 在竞技游戏领域，低于70%的好评率意味着社区口碑需要持续修复。好评率是滞后指标——即使最近的版本改善了体验，历史差评不会消失。

### 漫威争锋能否成为长青游戏？

这是决定网易未来3-5年增长故事的关键问题。

**支持长青化的证据：**

1. **漫威品牌的持续吸引力。** 漫威电影、漫画、动画不断推出新角色和新故事——每一个新角色都是游戏内容更新的天然素材。只要漫威品牌不衰落，游戏就有持续的内容来源。

2. **团队进化类游戏的特殊优势。** 漫威争锋是6v6团队射击游戏——这个品类的代表作（如守望先锋、无畏契约）都证明了"英雄/角色持续更新"的模式可以维持多年的用户粘性。

3. **同时在线已触底回升。** 从9.6万回升到20万+，趋势是向上的。如果能稳定在15-20万以上的同时在线，就具备了长期运营的基础。

**反对长青化的证据：**

1. **63%好评率是硬伤。** 竞技游戏极度依赖社区口碑——一旦被贴上"差评多"的标签，新玩家的进入意愿会显著降低。修复口碑比修复产品更难。

2. **同时在线暴跌85%的记忆很难抹去。** 即使目前回升到20万+，市场和玩家都记得它曾经从64万跌到9.6万。这种"高开低走"的叙事会长期压制预期。

3. **品牌授权是双刃剑。** 网易不拥有漫威品牌——它拥有的是开发权。如果合约到期后迪士尼/漫威不续约，或者提高授权费用，游戏的经济模型可能完全改变。品牌授权风险在第04篇详细展开。

**合理预期**：漫威争锋大概率能维持为一款"中等成功"的长线游戏——年收入在$3-5亿级别，而非首月年化的$16亿级别。如果运营持续改善、新角色持续吸引玩家，有可能进入"小长青"行列。但要达到《英雄联盟》《无畏契约》那种级别的全球长青——目前的数据不支持这个预期。

## 三、海外困境：10%的天花板怎么突破？

### 现状

| 指标 | 网易 | 腾讯 | 差距 |
|------|------|------|------|
| 海外游戏收入 | ~90亿（估，占游戏10%） | ~774亿（占游戏32%） | ~8倍 |
| 全球化策略 | 自研出海为主 | 收购+自研+发行 | 完全不同 |
| 海外工作室 | **7个已关闭** | 多个成熟工作室运营中 | — |

8倍的海外收入差距，不是一两款游戏能追上的——这是两种完全不同的全球化路径导致的结构性差距。

**腾讯的路径：用钱买时间。** 收购Supercell（部落冲突）、控股Riot（英雄联盟）、投资Epic（堡垒之夜）——通过资本运作直接获得全球顶级产品和团队。

**网易的路径：自研出海。** 依靠自己开发的产品打全球市场——成本更低、但成功率也更低。7个海外工作室关闭就是这条路径代价的具体体现。

### 7个工作室关闭的教训

7个海外工作室关闭不是一个小数字——它说明网易在海外建立本地化研发团队的尝试遭遇了系统性困难。

困难的根源在于：**游戏研发的核心竞争力很大程度上取决于文化理解。** 网易在中国市场有极深的文化洞察力（梦幻西游23年的中国文化运营就是证明），但这种能力很难简单复制到欧美、日韩市场。

关闭工作室之后，网易的海外策略更多转向"在中国研发、面向全球发行"——漫威争锋就是这种模式。这降低了成本风险，但也限制了在海外市场"本地化创新"的可能性。

**反方观点**：7个工作室关闭也可以理解为"及时止损"——丁磊认识到海外本地化研发这条路走不通，果断关掉，集中资源做"中国研发+全球发行"。从资本配置角度看，这不一定是坏事。

## 四、小程序游戏：535亿市场的缺席

### 这是一个多大的问题？

2025年中国小程序游戏市场规模535亿元——这几乎等于网易云音乐+有道的全部营收。

而网易在这个市场上的存在感接近于零。

### 为什么缺席？

小程序游戏的主要平台是微信——而微信是腾讯的。

| 维度 | 解释 |
|------|------|
| 平台归属 | 微信小程序生态完全由腾讯控制 |
| 分发优势 | 腾讯系游戏天然获得更多推荐和流量 |
| 技术壁垒 | 小程序游戏的技术框架和原生游戏不同，需要专门团队 |
| 用户习惯 | 小程序游戏的用户更偏轻度、碎片化，与网易擅长的重度游戏基因不同 |

**这不是网易"不想做"——而是平台不在自己手里。** 就像一家餐厅再好，如果商场是竞争对手开的，位置分配上天然吃亏。

### 影响有多大？

535亿市场，如果按照腾讯的利润率估算，可以贡献超过100亿的利润。网易吃不到这块蛋糕，等于每年少赚百亿级别的利润——而且这个市场还在高速增长。

**更重要的是趋势**：小程序游戏的增速远高于传统手游和端游。如果未来5年小程序游戏市场增长到1000亿+，网易的缺席将变成一个越来越大的结构性劣势。

**反方观点**：小程序游戏的利润率可能不如传统手游——平台抽成、用户付费意愿低、生命周期短。网易专注于高ARPU的重度游戏，利润率可能反而更高。缺席小程序游戏不一定是坏事——关键看总利润能不能增长。

## 五、版号：一个正在改善的外部环境

2025年，中国游戏版号发放量达到**1,771个——七年新高**。

| 年份 | 版号发放数 | 环境 |
|------|----------|------|
| 2021年 | 755 | 版号寒冬 |
| 2022年 | 512 | 最严厉的一年 |
| 2023年 | 1,076 | 开始恢复 |
| 2024年 | ~1,300（估） | 持续恢复 |
| 2025年 | **1,771** | **七年新高** |

版号发放量的恢复对网易是明确的利好——更多版号意味着更多新游戏可以上线。但版号利好对所有游戏公司都一样——网易能否从中获得超额收益，取决于自身的产品储备。

## 六、小结：游戏帝国的成色

把这一篇的分析总结成几句话：

1. **长青产品是网易游戏的核心护城河**——梦幻西游23年创纪录、永劫无间四周年登顶Steam——这种让游戏"活很久"的能力在全球罕见
2. **漫威争锋是最大变量**——首月惊艳但同时在线暴跌85%后回升至20万+，大概率是"中等成功"而非"超级长青"
3. **海外是最大的短板**——10%占比、7个工作室关闭、与腾讯8倍差距——出海之路任重道远
4. **小程序游戏535亿市场的缺席是结构性劣势**——平台不在自己手里，短期内无法改变
5. **版号环境正在改善**——1,771个创七年新高，但利好全行业而非网易独享

**如果用一句话概括网易的游戏帝国：国内极强、海外极弱、增长靠新品和出海——而新品和出海恰恰是最不确定的部分。**
