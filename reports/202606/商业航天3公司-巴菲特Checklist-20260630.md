## 一、铖昌科技（001270）

### 第一关：我能理解这门生意吗（能力圈）

- [x] **一句话赚钱逻辑**：设计星载相控阵T/R芯片，按颗卖给卫星制造商（每颗卫星需数百至千颗），独供国内低轨卫星星座
- [x] **10年后做什么**：大概率仍做T/R芯片，覆盖星载→机载→地面→低空经济
- [x] **关键变量**：星座组网速度（星网+千帆发射节奏）、竞争者是否出现、芯片制程迭代
- [x] **认知来源**：公开财报+行业研报+公告，非深度一手调研

> 段永平："这是个好生意——你只需要问：中国要不要建卫星互联网？如果要，就必须买他的芯片。"

**评分：★★★★☆**

**给竞争对手100亿能复制吗？** 高轨不能（频轨先占原则）。但低轨星座提供了另一条路——1000亿可以建一个替代系统。

**评分：★★★★☆**  
牌照+频轨壁垒极高,短期不可颠覆。但5-10年维度看,低轨替代是真实威胁。
