## 第二步：生意本质分析 — 段永平"对的生意"

### 盛弘股份：电力电子"瑞士军刀"

**一句话定义**：以电力电子模块化技术为底座，同时切入充电桩、储能PCS、电能质量、电池检测四个赛道的平台型设备公司。

**商业模式**：硬件销售为主（设备+模块），项目制交付+少量运维服务收入。一次性销售为主，非订阅模式。

**收入漏斗（2025年）**：
```
[充电桩设备出货量×ASP] + [储能PCS出货×ASP] + [APF/SVG出货×ASP] + [电池检测设备×ASP]
    │ 合计
[营收 34.63亿元]
    │ - 直接成本（原材料：功率半导体IGBT/SiC、磁性器件、PCB、结构件 ≈ 21.3亿）
[毛利 13.33亿元]（毛利率38.47%）
    │ - 运营费用（销售1.87亿 + 管理1.35亿 + 研发2.94亿 + 财务-1.08亿 = 5.08亿）
[营业利润 ~5.5亿元]
    │ - 所得税等
[净利润 4.76亿元]（净利率13.70%）
    │ + 折旧/非现金项目 - 资本支出
[自由现金流 ~7.0亿元]（FCF/净利润 ≈ 147%，现金转化优秀）
```

**核心漏斗观察**：
- 终端FCF转化率极高（7.0亿FCF / 34.63亿营收 = 20.2%），说明轻资产模式运转良好
- 毛利率分化巨大：电能质量58.5% vs 储能27.9%，意味着业务结构决定了盈利质量
- 2025年经营现金流大幅改善（+92%），说明此前的应收账款问题正在缓解

### 天承科技：PCB化学品"卖水人"

**一句话定义**：为PCB/半导体制造商提供核心生产化学品（沉铜液、电镀液）的"卖水人"公司，受益于AI算力PCB价值量爆增。

**商业模式**：耗材型销售（配方+化学品持续供应），客户包线后形成较强粘性，类似"剃须刀+刀片"模型。

**收入漏斗（2025年）**：
```
[客户PCB产线包线数 × 单线年用药量 × 药水单价]
    │ 合计
[营收 4.71亿元]
    │ - 直接成本（原材料：铜盐、络合剂、添加剂、包装 ≈ 2.83亿）
[毛利 1.88亿元]（毛利率40.04%）
    │ - 运营费用（销售0.22亿 + 管理0.28亿 + 研发0.34亿 + 财务-0.02亿 = 0.82亿）
[营业利润 ~1.06亿元]
    │ - 所得税等
[净利润 0.87亿元]（净利率18.35%）
    │ + 折旧/非现金 - 资本支出
[自由现金流 ~0.61亿元]（FCF/净利润 ≈ 70%）
```

**核心漏斗观察**：
- 净利率18.35%高于盛弘（13.70%），说明"卖水人"生意本质更优
- 但FCF转化率偏低（主要因应收账款占净利润196%），客户付款慢
- 整体体量太小（4.71亿营收），在全球PCB化学品市场（约250亿美元）中份额极低

**段永平式追问**：
- 盛弘的生意好在哪？→ 电力电子技术底座通用性强，四条赛道分散风险，每个赛道都处于结构性增长期
- 天承的生意好在哪？→ 耗材粘性+国产替代确定性，一旦通过客户认证就难以被替换，但体量太小还远未到"好生意"的阶段

## 第六步：行业与文明趋势 — 李录"文明演进框架"

### 盛弘所处的行业

**核心趋势**：全球电气化（电动车+储能+数据中心）是文明级别的范式转移
- 电动车充电：2027年中国目标2800万个充电设施，需求确定
- 储能：全球新能源并网+电网稳定性需求，2030年中国目标3.7亿千瓦装机
- AIDC：全球四大云厂2026年资本开支6000-6500亿美元，电力是核心瓶颈

**盛弘在价值链中的位置**：中游设备供应商，不掌握核心材料（功率半导体），也不掌握终端运营

### 天承所处的行业

**核心趋势**：AI算力爆发 → PCB层数/复杂度飙升 → 电子化学品用量倍增
- GB300/Rubin AI服务器：单机PCB价值量是传统服务器的3-5倍
- mSAP工艺升级：水平沉铜液用量+30%，电镀添加剂需求近翻倍
- 国产替代：高端PCB化学品80%被安美特等外资垄断，国产化率仅15-25%

**天承在价值链中的位置**：上游核心材料供应商，掌握配方Know-how，地位类似于"半导体光刻胶之于芯片制造"

**李录式追问**：20年后回看？

- 天承：如果成功从PCB化学品拓展到半导体封装化学品，有可能成为"中国的安美特"——但目前还太早
