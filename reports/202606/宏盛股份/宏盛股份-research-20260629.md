## 第二步：生意本质分析 — 段永平"对的生意"

### 一句话定义

**宏盛股份是一家铝制板翅式换热器的定制化OEM配套商，传统业务平庸但稳定，正通过合资公司押注数据中心液冷这一爆发性赛道。**

### 收入结构（FY2025）

| 业务板块 | 营收（万元） | 占比 | 同比增速 | 毛利率（估） |
|----------|-------------|------|---------|-------------|
| 换热器（传统） | 69,783 | 91.4% | +9.85% | ~27% |
| 锂电池PACK | 3,295 | 4.3% | -3.21% | 低（已终止境外电商） |
| 热管理系统 | 870 | 1.1% | +19.62% | ~25% |
| 深冷设备 | 135 | 0.2% | -90.56% | — |
| 其他 | 2,302 | 3.0% | +3.76% | — |
| **合计** | **76,385** | **100%** | **+7.13%** | **25.68%** |

**注**：液冷CDU业务通过合资公司和宏智（持股49%）开展，**不体现在母公司营收中**，仅通过投资收益反映。

### 5年盈利能力趋势

| 年度 | 营收（亿元） | 净利润（万元） | 毛利率 | 净利率 | ROE |
|------|-------------|---------------|--------|--------|-----|
| FY2021 | ~5.7（估） | ~1,500（估） | ~23% | ~2.6% | ~2.8% |
| FY2022 | ~7.14 | ~5,300 | ~19.2% | ~7% | ~9.6% |
| FY2023 | 6.96 | 5,375 | 24.9% | ~8% | ~9.4% |
| FY2024 | 7.13 | 4,936 | 26.0% | 7.0% | 8.7% |
| FY2025 | 7.64 | 8,239 | 25.7% | 10.7% | 13.7% |

**趋势判断**：
- 营收5年复合增速仅~6%，增长乏力
- 净利润波动巨大（FY2020仅550-650万 → FY2025的8,239万），抗周期能力弱
- FY2025净利润大增66.9%，但主因是终止亏损的境外锂电池PACK业务（减亏~3,000万），**并非核心业务大幅改善**
- 毛利率近3年稳定在25-26%，在换热器行业中偏高，反映定制化产品的附加值

### 商业模式画布

```
客户（OEM配套）：阿特拉斯科普柯、西门子、三一重工、康明斯、利勃海尔等
         │
    定制化换热器设计+生产（非标品）
         │
收入模式：一次性销售，项目制，无订阅/复购锁定
         │
价值主张：10,000+种型号的定制化能力 + 国际认证（ASME/CE）
         │
关键资源：真空钎焊工艺 + 风洞实验系统 + 20年行业积累
```

**商业模式评价**：这是一门**典型的制造业OEM配套生意**——客户粘性来自认证和定制化成本，而非网络效应或品牌忠诚。一旦通过客户认证（如谷歌审厂），短期内不会被替换，但长期来看壁垒并不深。

### 收入漏斗分析

```
[传统换热器业务量]
  产能：约8-10万台换热器/年（估）
  × 平均单价：约700-900元/台（估）
    │
[营收：7.64亿元]
    │ - 直接材料成本（铝材/铜材约占60%） ≈ 4.57亿
    │ - 制造费用 ≈ 1.11亿
[毛利：1.96亿（毛利率25.68%）]
    │ - 研发费用 0.34亿（4.4%）
    │ - 销售费用 ~0.25亿
    │ - 管理费用 ~0.50亿
    │ - 财务费用 ~0.10亿
[营业利润：~0.87亿]
    │ - 所得税 ~0.10亿
    │ + 投资收益（含和宏智）
[净利润：0.82亿（净利率10.7%）]
    │ + 折旧摊销 ~0.35亿
    │ - 资本支出 ~0.15亿
[自由现金流：~1.02亿（FCF/Revenue ≈ 13.4%）]
```

**漏斗核心发现**：
1. 材料成本占比约60%，铝价波动直接影响毛利率——这不是一门"轻资产"生意
2. 净利率10.7%在A股制造业中属中等偏上，但ROIC历史中位数仅6.96%，说明**资本效率不高**
3. FCF/Revenue约13.4%，经营现金流>净利润，现金流质量尚可
4. 关键缺失：**液冷CDU的漏斗不在母公司报表中**——这是最大的信息盲区

### 段永平式追问

> **"这门生意好在哪？用一句话。"**

**答**：传统换热器业务是一门"能活但不够好"的生意——毛利率25%、ROIC不到7%、增长缓慢，不是段永平会喜欢的"好生意"。真正让市场兴奋的是液冷CDU的潜力，但这个"好生意"是通过一家持股49%的合资公司来做的，**好处到底有多少能落到上市公司口袋里，是最核心的问题**。

## 第三步：护城河评估 — 巴菲特"经济护城河"

| 护城河类型 | 评分 | 证据 |
|-----------|------|------|
| 品牌/定价权 | ★★☆☆☆ | 无终端品牌认知，OEM配套商定价权有限。毛利率26%高于部分同行，但更多来自定制化，非品牌溢价 |
| 转换成本 | ★★★☆☆ | 通过客户认证（ASME/CE/谷歌审厂）后有一定粘性，更换供应商需重新认证（6-12个月）。但换热器非客户核心系统，替换动力低但替换成本也不极高 |
| 网络效应 | ★☆☆☆☆ | 无。制造业OEM配套不存在网络效应 |
| 规模效应 | ★★☆☆☆ | 年营收仅7.6亿，在全球换热器市场中是小玩家。规模不足以形成显著的成本优势 |
| 技术/专利壁垒 | ★★☆☆☆ | 78项专利（vs 银轮685项），专利数量偏少。10,000+种定制化型号是核心know-how，但更多是工程经验积累而非不可复制的技术 |

**护城河总评：★★☆☆☆（窄护城河/无明显护城河）**

### 护城河趋势

- **过去5年**：护城河基本持平。传统业务客户关系稳定，但未见明显拓宽
- **未来5年预判**：
  - 如果液冷CDU业务成功，可能通过进入Meta/谷歌供应链建立新的**客户壁垒**
  - 但液冷业务通过合资公司开展，护城河的受益方是和宏智而非宏盛母公司
  - 加拿大建厂可能增强海外客户的供应链安全感，但短期是成本负担

### 巴菲特式追问

> **"10年后这条护城河还在吗？什么能摧毁它？"**

**答**：传统换热器业务的护城河（定制化+认证）10年后大概率还在，但也不会变宽。能摧毁它的是：(1) 铝制板翅式换热器被新技术替代（如微通道、3D打印换热器），(2) 中国制造业整体产能过剩导致价格战。液冷CDU的护城河取决于和宏智能否建立独立于和信精密的客户关系——如果始终依赖合资方的渠道，护城河归属权存疑。

### 偏误自查

- [x] **叙事偏差**：液冷赛道的宏大叙事极具吸引力，容易忽视执行层面的困难（合资公司控制权、品质问题、竞争格局）
- [x] **锚定效应**：股价从99元跌至52元，容易产生"已经跌了一半，便宜了"的错觉——但如果液冷不达预期，按传统业务估值仅值15-20元
- [x] **幸存者偏差**：只看到液冷赛道的赢家（英维克），没有看到众多切入液冷但未能成功的小公司

### 空方核心论点

1. **估值泡沫**：按传统业务净利1亿元计算，PE>50x，严重透支。市场定价100%基于液冷预期
2. **49%持股的合资公司是定时炸弹**：不并表、不可控，好处有限、风险无限
3. **2026Q1已经"见光死"**：和宏智Q1投资收益仅2.18万元，与市场预期的上亿元天差地别
4. **竞争劣势明显**：英维克市值200亿+、飞龙市值100亿+，宏盛52亿的体量在液冷赛道是"小虾米"
5. **ROIC仅6.96%**：传统业务本身就不是好生意，不值得高估值

> **"我最可能在哪里犯错？聪明人为什么会不买/做空这家公司？"**

**答**：最容易犯的错是**把合资公司的增长当成上市公司的增长**。和宏智即使做到16亿收入，宏盛只能按49%权益法确认投资收益（税前约1.5亿×49%×净利率）。聪明人不买的理由很简单：(1) 传统业务不值这个价，(2) 液冷业务的控制权不在自己手里，(3) 公司治理历史有瑕疵。

## 第六步：行业与文明趋势 — 李录"文明演进框架"

### 所在行业是否处于"文明级范式转移"？

**传统换热器业务：否。** 这是一个成熟的制造业细分领域，增长与GDP和工业化进程同步，没有范式转移。

**数据中心液冷：是。** AI算力爆发驱动的散热革命是一个真实的**文明级范式转移**：
- 英伟达Rubin架构芯片功耗达2,300W，单机柜功率密度100kW+
- 风冷极限仅40kW/机柜，液冷是唯一解决方案
- 2026年全球液冷市场规模预计突破1,000亿元
- 液冷渗透率从个位数快速上升至35-50%

### 历史类比

| 维度 | 当前AI散热革命 | 历史类比：PC时代的散热 |
|------|--------------|---------------------|
| 驱动力 | GPU功耗指数级增长 | CPU频率提升（奔腾4时代的风扇噪音问题） |
| 解决方案 | 液冷取代风冷 | 热管+风扇取代简单散热片 |
| 产业机会 | CDU、冷板、快接头、冷却液 | 散热模组、热管、风扇 |
| 赢家特征 | 早期切入大客户供应链的公司 | 台达电、奇鋐 |

### 宏盛在产业价值链中的位置

```
AI芯片（英伟达/谷歌） → 服务器OEM（广达/戴尔） → 液冷系统集成（和信精密等）
                                                        │
                                              CDU换热器（和宏智/宏盛）
```
